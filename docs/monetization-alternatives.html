AdSense 심사가 빡세도 (또는 통과해도) 병행 가능한 7개 카테고리 대안. 한국 시험 학습 사이트(gcnote)의 단계별 우선순위 정리.

왜 이 문서가 필요한가

한국 사이트는 AdSense 심사가 보통 1~3개월 걸리거나 탈락한다

한국 신생 사이트가 자주 거절되는 이유:

  • 도메인 만든 지 6개월 미만
  • 콘텐츠 30개 미만 또는 AI 생성 의심
  • 중복/얇은 콘텐츠 (자동 생성 페이지 많음)
  • Privacy Policy / Terms 미흡
  • 트래픽 너무 적음 (월 1만 PV 미만)

핵심 깨달음 1: AdSense RPM ₩2K vs Affiliate CPL ₩3~10K. 학원 affiliate 1건이 AdSense 한 달치보다 큼. AdSense는 보너스, Affiliate가 본업.

핵심 깨달음 2 (사용자 판단): 학원 BD는 트래픽이 커진 후에. 월 PV 35K로 NextStudy 메일 보내면 협상력 0 — 단가 후려쳐지거나 무시당함. 트래픽 키우기 → 협상력 확보 → 본격 BD 순서.

현재 상태 (2026-04-26): AdSense 심사 신청 완료 (4/25) → 결과 대기 중. 카카오 애드핏을 병행 신청해 한국 광고주 base 확보 (한국 사용자 100% → 카카오가 우리 케이스에 더 자연스러움).

왜 이 순서인가

학원 입장: 월 PV 100K 미만 = "주목할 만한 사이트 아님". 단가 후려쳐도 lead 양 적어 의미 없음. 월 PV 100K+ = 한 달 ₩10~30만 수익 보장 = 미팅·계약 가치. 500K+ = banner sponsorship 협상 권력 우리에게.

📈 광고비 max 사용 원칙 (2026-04-26 추가)

한국 학원 부재 채널 (Google·인스타·스레드·카카오)은 ROAS 6× 가정 가능. 광고비 ₩30만 → ₩100만 → ₩300만 → ₩1,000만 → ₩2,000만 (max)까지 단계 확장. ROAS > 4× 유지되는 한 광고비 늘리는 게 합리적. 한도는 channel capacity (Google·인스타 일 노출 한계 ≈ 월 ₩1,500~2,500만). v3 §05 ⑤-2 광고비 별 매출 시뮬 참조.

왜 Ezoic은 보류로 변경했는가

초기엔 1순위로 평가했지만 우리 사이트 분석 결과 (1) 한국 사용자 100% → Ezoic의 글로벌 광고주 base 강점이 무의미, (2) 페이지 속도 -15~30% 손상 → 우리는 SEO organic 30% 의존하므로 GSC 순위 하락 위험, (3) 정산이 PayPal/USD → 환전 수수료. 카카오 + AdSense 정착 후 1~2개월 A/B 테스트로 net gain 확인 후 결정.

한국 시험 사이트의 진짜 수익은 학원 affiliate (CPL ₩3K~10K). 광고 RPM은 보너스. 단, 월 PV 35K로 학원 메일 보내면 협상력 0 — 단가 후려쳐짐. 이번주는 트래픽 키우기 (9급 노트 + 인스타·스레드 일일) + 카카오 애드핏 신청 + 쿠팡·응원결제만. AdSense는 4/25 신청 완료 → 결과 대기. 학원 BD는 50K 시드 컨택 → 100K 본격 미팅 순서.
