# 기출노트 마케팅 전략 (Single Source of Truth)

> 이 문서는 마케팅 에이전트(`.claude/agents/marketing.md`)가 **항상 먼저 읽는** 뇌다.
> 의사결정·실행 전에 여기 맥락을 깔고 시작한다. 세션마다 "처음 보는 것처럼" 굴지 않기 위함.
> 최종 갱신: 2026-06-22

## 0. 한 줄 요약

> **콘텐츠는 끝까지 무료·공개(SEO 플라이휠 연료), 돈은 로그인 뒤 "합격 도구"에서 받는다.
> 하드 페이월 절대 금지. 수익 최적해 = SEO 최적해라 트레이드오프가 없다.**

## 1. 비즈니스 컨텍스트

- **제품**: 기출노트 한능검 — 한국사능력검정시험 학습 플랫폼 (gcnote.co.kr)
- **런칭**: 2026년 3월 말 (이 문서 기준 사이트 나이 ≈ 3개월)
- **확장**: 공무원 시험으로 확장 중 (한능검 외 두 번째 축)
- **시험 캘린더 = 트래픽 캘린더**
  - 한능검: **연 4회** → 1년의 약 절반이 성수기
  - 공무원: **4월** (주요 시험) → 6월~3월이 "비수기"가 아니라 공무원 시즌 빌드업
  - 결론: **연중 수요가 분산됨. 죽는 비수기가 없음 = 구독 모델에 이상적**
- **성장 방식**: **광고비 0원**, 순수 SEO + 바이럴
  - → CAC ≈ 0. 어떤 과금이든 고마진. 유기적 PMF가 이미 검증됨

## 3. SEO 플라이휠 (이 사업의 진짜 모트)

런칭 **1달 만에 네이버 첫 페이지** 진입 — 네이버 블로그도 안 하고. 원인 조합:
1. **틈새 공백**: 한능검 *해설*은 네이버 블로그/카페/지식iN이 약한 영역. 구조화 DB 사이트가 통째로 점유
2. **롱테일 폭격**: 회차별·문제별 1,900+ 페이지가 "78회 12번 해설" 같은 구체 의도를 1:1로 흡수
3. **engagement 신호**: 41 PV/user, 17분 = 네이버/구글이 "좋은 페이지"로 읽는 직접 신호 → 빠르게 + 계속 순위 밀어올림

```
높은 체류시간 → SEO 순위↑ → 유입↑ → engagement↑ → 순위 더↑   (광고비 0으로 도는 바퀴)
```

**→ 이게 "콘텐츠 절대 페이월 금지"의 결정적 근거.**
콘텐츠를 막거나 유저를 짜증나게 하면 체류시간↓ → 순위↓ → **플라이휠 역회전 = 자해.**

## 4. 수익화 모델: 프리미엄(Freemium), 하드 페이월 ❌

검색엔진이 색인하는 것과 소비자가 돈 내는 건 **다른 레이어**라 충돌하지 않는다.
- 색인 대상 = 정적·공개·페이지 단위 **콘텐츠**
- 과금 대상 = 동적·개인별·로그인 뒤 **도구** (크롤러가 색인하고 싶어도 못 함)

### 무료/유료 경계선

| 무료·공개 (SEO 엔진, 끝까지 오픈) | 유료 (계정 뒤, 색인 안 됨) |
|---|---|
| 기출 문제 + 정답 + **해설 본문** | 오답노트 **클라우드 동기화**·기기 연동 |
| 요약노트 | 내 기록·약점 분석 리포트 |
| 키워드 페이지 | **맞춤 모의고사 생성** / 약점 집중 세션 |
| 시대·유형별 둘러보기 | 망각곡선 복습 스케줄 / D-30 집중코스 |
| (선택) 무료 풀이 1~2개 | 광고 제거 · PDF·인쇄 다운로드 · 합격 예측 |

핵심: 파는 건 *정보*가 아니라 **시간·합격률**. 공무원층이 공단기에 돈 쓰는 이유가 이것.

## 5. SEO 안 깎는 규칙

1. **콘텐츠 소프트 페이월 절대 금지** — 클로킹(구글엔 보이고 유저엔 가리기)은 가이드 위반 + 순위 폭락. 콘텐츠는 진짜로 다 공개
2. **유료 기능 = 로그인 뒤 = 이미 robots.txt 차단 영역**(`/study/session`, `/my-record`). 색인 안 되는 게 정상
3. **AdSense는 무료 티어에만, 혹은 버린다** — 거슬리는 광고는 체류시간을 깎아 SEO 플라이휠을 역회전시킴. AdSense 거절 사유였던 "가치 낮은 콘텐츠"는 해설 밀도 강화로 해결(= SEO에도 직접 이득). 깔끔한 유료 모델이 수익·SEO 둘 다에 유리

### 네이버는 구글과 다르게
- 자체 크롤러(Yeti) + 블로그/카페/지식iN 편애 → **본문 텍스트 풍부 + 신선도**를 더 봄
- 해설·요약을 **네이버 블로그로 syndication**(요약 + 원문 링크)하면 별도 유입
- 서치어드바이저 등록 완료 → 검색노출/클릭 리포트로 순위 키워드 추적

## 6. 인프라 현황 & 갭

- **계정 시스템 없음** — 오답노트·기록 전부 localStorage (기기 바꾸면 소실). **유료화의 전제 자체가 없음** → 1순위 작업
- **GA4 = 클라이언트 추적만** (gtag write). `question_answered` 커스텀 이벤트 미설치 → WAL은 users(7일) proxy. 데이터 *읽기*(Data API)는 서비스계정 별도 필요 (`scripts/ga4/` 도구 준비됨, 키만 꽂으면 됨)
- **데이터 저장소 지도** (헷갈리지 말 것):
  - 기출 데이터 → **R2 공개 버킷** `korea-history-data` (`pub-777d764118c647a2a715fb0b17072419.r2.dev`), `manifest.json`이 소스. 키 없이 fetch 가능
  - 한능검 DB(게시판/배너) → Supabase `uonznnypdnerdigfyfci` (CLAUDE.md 기재)
  - ⚠️ 현재 연결된 Supabase MCP는 **다른 계정**(`fxzwigjkbsptvsjraqwa`/tangobook, 아동 단어앱+마케팅도구). 한능검 프로젝트 권한 없음
  - 공무원 콘텐츠는 위 어디에도 안 보임 — 위치 별도 확인 필요
