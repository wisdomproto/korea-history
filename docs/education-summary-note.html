1. 교육심리 (발달·학습·동기)

TOPIC 0132회

피아제 (J. Piaget) 인지발달 4단계

  • 감각운동기 (0~2): 대상영속성
  • 전조작기 (2~7): 자기중심성·물활론·직관적 사고
  • 구체적 조작기 (7~11): 보존·서열·분류·가역성
  • 형식적 조작기 (11+): 추상·연역·가설

에릭슨 (E. Erikson) 심리사회 8단계

단계위기
0~1신뢰 vs 불신
1~3자율성 vs 수치심
3~6주도성 vs 죄책감
6~12근면성 vs 열등감
12~18정체성 vs 역할혼미
20대친밀성 vs 고립
중년생산성 vs 침체
노년통합 vs 절망

마샤 (Marcia) 정체성 지위

  • 정체성 성취: 위기 + 전념
  • 정체성 유실 (조기완료): 위기 X + 전념 (부모 가치 수용)
  • 정체성 유예: 위기 + 전념 X (탐색 중)
  • 정체성 혼미: 위기 X + 전념 X

지능 이론

  • 스피어만 g요인·서스톤 다요인설·가드너 다중지능 (8가지) ·스턴버그 삼원지능 (분석·창의·실용)
  • 유동성지능 vs 결정성지능 (Cattell): 유동성 — 새로운 문제해결, 노화 ↓ / 결정성 — 학습된 지식, 노화 ↑

동기이론

  • 매슬로우 욕구 5단계·맥그리거 X·Y이론
  • 와이너 귀인이론: 능력·노력·과제난이도·운 4가지. 노력 = 내부·불안정·통제가능
  • 켈러 ARCS: 주의(Attention)·관련성(Relevance)·자신감(Confidence)·만족(Satisfaction) 4요소

2. 교육행정 (조직·인사·재정)

교육행정 원리

  • 적도집권: 중앙집권 + 지방분권 균형

학교조직

  • 참모조직 (Staff): 자문·기획 — 교무부·연구부·생활지도부 등
  • 계선조직 (Line): 명령·집행 — 교장 → 교감 → 부장교사 → 담임
  • 이중조직: 학교는 관료제 + 전문직 조직 결합
  • 이완결합 (Loosely Coupled): 와익 — 학교는 부서 간 결합도 약함

호손 실험 (Hawthorne Experiment)

메이요·뢰슬리스버거 — 인간관계론 등장. 비공식 조직·심리적 만족·집단 사기가 생산성에 영향. 신고전적 조직이론 시발점.

교육비

  • 공교육비: 정부·자치단체 부담 — 일반회계·교육특별회계·교육비특별회계
  • 사교육비: 학부모·학생 부담 — 학원비·과외비·교재비
  • 직접교육비 vs 간접교육비: 직접 — 등록금·교재 / 간접 — 기회비용·교통

지방교육재정교부금

국세 (내국세) 20.79% — 시·도 교육청에 교부. 보통교부금 + 특별교부금. 교육의 지역 간 격차 해소.

6. 교육사회학

기능론 vs 갈등론

기능론 (Durkheim·Parsons)갈등론 (Bowles·Gintis)
교육 기능사회화·선발·통합지배계급 재생산·이데올로기
학교능력주의 — 공정 분배계층 재생산 도구
대표 학자뒤르켐·파슨스·머튼보울즈·진티스·일리치·프레이리

주요 학자·이론

  • 번스타인 (Bernstein): 계층과 언어사용 — 제한된 어법 (working class) vs 정교화된 어법 (middle class). 학교는 정교한 어법 사용 → 노동계급 불리
  • 부르디외 (Bourdieu): 문화자본·아비투스 — 가정 환경의 문화자본이 학업 성취에 영향
  • 일리치 (Illich): 탈학교사회 — 학교는 진정한 학습 방해
  • 프레이리 (Freire): 은행저축식 교육 비판 → 문제제기식 교육
  • 로젠탈·제이콥슨: 피그말리온 효과 — 교사 기대가 학생 성취에 영향
  • 콜만 보고서: 학교의 한계 — 가정·또래 영향이 더 큼

평가의 양호도

  • 타당도 (Validity): 측정하려는 것을 측정 — 내용·준거·구성·예측·공인
  • 신뢰도 (Reliability): 일관성 — 검사재검사·동형검사·반분·내적일관성 (KR-20·크론바흐 알파)
  • 객관도 (Objectivity): 채점자 간 일치도
  • 실용도 (Usability): 시간·비용·실시 용이성
