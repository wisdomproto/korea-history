1. 컴퓨터 일반 — 하드웨어·소프트웨어

컴퓨터 5대 장치

  • 입력: 키보드·마우스·스캐너·터치스크린·OMR/OCR/MICR/바코드
  • 출력: 모니터(CRT·LCD·LED·OLED)·프린터(잉크젯·레이저·도트)·플로터·스피커
  • 기억: 주기억(RAM·ROM)·보조기억(HDD·SSD·USB·CD/DVD/Blu-ray)·캐시·레지스터
  • 제어: CPU 내부, 명령 해석·실행 제어
  • 연산: ALU, 사칙·논리

중앙처리장치 (CPU)

  • 구성: 제어장치 + 연산장치 + 레지스터
  • 주요 레지스터: PC(프로그램카운터)·IR(명령)·MAR(주소)·MBR/MDR(데이터)·AC(누산기)
  • 성능: 클럭 (Hz·GHz)·MIPS·FLOPS
  • 병렬처리: 파이프라인·슈퍼스칼라·멀티코어

기억장치 계층

  • 레지스터 → 캐시(L1·L2·L3) → 주기억 → 보조기억 → 백업
  • RAM: SRAM(빠름·캐시) vs DRAM(주기억)
  • ROM: Mask·PROM·EPROM·EEPROM·플래시
  • SSD: 플래시 메모리 기반·HDD보다 빠름·기계적 부품 X
📌 출제: CPU 레지스터 + 기억장치 계층.

8. Access — 데이터베이스

  • DML: SELECT·INSERT·UPDATE·DELETE
  • DCL: GRANT·REVOKE
  • SELECT: SELECT 필드 FROM 테이블 WHERE 조건 GROUP BY 필드 HAVING 조건 ORDER BY 필드
  • JOIN: INNER·LEFT·RIGHT·FULL OUTER
  • 집계: COUNT·SUM·AVG·MAX·MIN + GROUP BY

쿼리

  • 선택 쿼리: 데이터 조회
  • 크로스탭 쿼리: 행·열·집계 (피벗처럼)
