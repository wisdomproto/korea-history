# 멤버십 / 유료화 모델 — 설계 결정 (2026-06-24)

## 한 줄 요약
기존 기능 그대로 두고 **듀오링고형 F2P**로 수익화한다. 무료 = 광고 + 한도, 프리미엄 = **"광고 없이 무제한"**. 새 기능은 만들지 않는다.

## 데이터가 결정을 이끌었다 (GA4/GSC 실측, 90일)
- **유입의 ~47%가 네이버** (m.search.naver 25.8% + naver organic 19.7% + kin/cafe 등). 구글 organic은 **3.3%**(브랜드 'gcnote/기출노트'만 클릭, 콘텐츠 키워드 ~0). Bing 7.4%(구글의 2배). Direct 30%(브랜드·재방문 강함).
- **organic 랜딩(검색 진입) 분포**: 홈 32.4% · **/notes 28.3%** · other(공무원 랜딩) 20.4% · /exam 8.9% · /study 6.6%.
  → **요약노트가 문제 페이지보다 검색 진입 3배.** 개별 노트(s1-01·s1-04…)가 랜딩 top. "노트=약한 롱테일"은 **데이터가 반증**.
- 성장 트리거: 디시인사이드 수동 글 + **체류 19분**(네이버 D.I.A.+ 만족신호)으로 비싼 키워드를 1달 만에 확보. → 검색 랭킹은 *벌어들인* 자산이라 함부로 deindex 금지.

## 티어
| | 무료(로그인) | 프리미엄 |
|---|---|---|
| 기출 풀이 + AI 해설 | 하루 한도 + 광고 | 무제한 · 광고 없음 |
| 요약노트 | **미리보기**(전체는 로그인) | 전체 |
| 문제↔노트 연결 | 미리보기(로그인 유도) | ✓ |
| 오답노트 | 한도 + 광고 | 무제한 |
| 기록 클라우드 동기화 | ✓(무료) | ✓ |
| 광고 | 있음 (+보상형으로 한도 충전) | 없음 |

- 비로그인(게스트): 기출·해설·노트 **미리보기**까지 가능 → 검색 유입·SEO 유지.
- **가입 즉시 프리미엄 7일 무료 체험**(카드 X) → 만료 시 무료로 강등 + 페이월.
- **추천 = 양면**: 초대자 +7일 / 초대받은 친구도 +7일(=14일). 어뷰징 방어(이메일 인증+첫 세션 적립, 누적 상한, 기기 중복 차단).
- 로그인 = **카카오 + 구글**. 동기화는 무료(잔류 앵커).

## 가드레일 (안 하는 것)
1. **콘텐츠를 봇에서 가리지 않는다** — 게이팅은 "미리보기 + 로그인 월"로 (deindex = 28% 진입·랭킹 증발). 봇엔 풀/사람엔 프리뷰 = 클로킹이라 금지(둘 다 동일 프리뷰).
2. **마찰은 첫 세션에 세게 걸지 않는다** — 한도는 넉넉히 시작(콜드 첫 입질은 매끈 → 훅·체류·바이럴 보존), 헤비 유저가 막히게(F2P 정석).
3. **지저분한 광고 금지** — 보상형(opt-in) 광고는 OK, 수동 브라우징 침해성 광고는 X.
4. **새 기능 안 만든다** — AI Q&A·합격예측·SRS 등은 feature creep. 검증된 자산(기출·오답·노트)으로 충분.

## 결제
- PG = **토스페이먼츠 + 카드 자동결제(빌링)** (Stripe는 한국 사업자 불가). 카카오/네이버페이 정기결제 필요해지면 포트원 검토.
- 데이터 모델 핵심: **`premium_until` 타임스탬프 1개**로 트라이얼+추천+결제 통일. `isPremium = premium_until > now`.

## 빌드 순서 (각자 독립)
1. **인증** (Supabase Auth + 카카오/구글 + localStorage→계정 마이그레이션 + 동기화) ← 전제조건
2. 구독·결제(토스 빌링) + 7일 트라이얼 + 양면 추천 + 페이월
3. F2P 마찰(광고 + 한도) + 요약노트 미리보기 게이팅
4. (파킹) 푸시 알림 — auth 위에. SRS 데일리 알림/트라이얼 만료/추천에 사용
