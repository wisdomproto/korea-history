1. 회계의 기본 — 거래·계정·복식부기

회계 등식

  • 자산 = 부채 + 자본 (재무상태표 등식)
  • 수익 - 비용 = 순이익(순손실) (손익계산서)
  • 자본 변동: 기초자본 + 추가출자 - 인출 + 순이익 = 기말자본

회계상 거래

  • 요건: ① 자산·부채·자본·수익·비용에 변동 ② 금액 측정 가능
  • 거래 X: 종업원 채용·계약·주문·견적·시세 변동·도난(미발견)·화재(발견 전)
  • 거래 O: 매매·차입·상환·받음·지급·발생·소멸

분개 (Journal Entry)

6. 원가회계 — 원가의 분류

원가의 3요소

  • 재료비: 직접재료비 + 간접재료비
  • 노무비: 직접노무비 + 간접노무비
  • 경비: 직접경비 + 제조간접비 (감가상각·임차료·전력·수도)

원가 분류

기준분류
제품과 관련성직접원가 vs 간접원가
조업도와 관계변동원가 vs 고정원가 vs 준변동·준고정
의사결정 관련관련원가 vs 비관련원가 / 매몰원가 / 기회원가
자산화 가능제품원가 (매출원가에 대응) vs 기간원가 (판관비)

제조원가 흐름

  • 재료 → 원재료 → 재공품 → 제품 → 매출원가
  • 제조원가: 직접재료비 + 직접노무비 + 제조간접비
  • 당기제품제조원가: 기초재공품 + 당기제조원가 - 기말재공품
  • 매출원가: 기초제품 + 당기제품제조원가 - 기말제품

제조간접비 배부

  • 예정배부율 = 제조간접비 예산 / 배부기준 (예: 직접노동시간)
  • 배부차이: 실제 vs 예정. 과대배부·과소배부
  • 차이 처리: 매출원가에서 조정 또는 비례 배분
📌 출제: 제조원가 흐름 + 제조간접비 배부 매년.

7. 원가계산 방법

개별원가계산 vs 종합원가계산

구분개별원가종합원가
대상주문 생산 (조선·건설·인쇄)대량 생산 (시멘트·정유·화학)
원가 집계제조지시서별공정별
완성품환산량불필요필요

완성품환산량 (종합원가)

  • 평균법: (기초재공품 + 당기착수 완성품) + 기말재공품 × 완성도
  • 선입선출법: 기초재공품 완성분 + 당기착수 완성품 + 기말재공품
  • 재료비·가공비 별도 계산 (투입 시점 다름)

공손·감손

  • 정상공손: 계획된 공손 → 양품 원가에 배부
  • 비정상공손: 당기 손실 처리

전부원가 vs 변동원가

  • 전부원가: 고정제조간접비도 제품원가에 포함 (외부보고용)
  • 변동원가: 변동제조간접비만 제품원가, 고정은 기간비용 (CVP 분석)

CVP 분석

  • 공헌이익 = 매출액 - 변동원가
  • 공헌이익률 = 공헌이익 / 매출액
  • 손익분기점 (BEP) 매출액 = 고정원가 / 공헌이익률
  • BEP 판매수량 = 고정원가 / 단위당 공헌이익
📌 출제: 완성품환산량 + 손익분기점 계산 매년.

9. 법인세·소득세 (전산회계 1급 범위)

법인세

  • 납세의무자: 영리내국법인·비영리내국법인·외국법인
  • 과세표준: 각 사업연도소득 - 이월결손금 - 비과세 - 소득공제
  • 세율 (2024): 2억 이하 9%, 2~200억 19%, 200~3000억 21%, 3000억 초과 24%
  • 신고·납부: 각 사업연도 종료일부터 3개월 이내

세무조정

  • 회계상 손익 → 세무상 익금·손금 차이 조정
  • 익금산입: 세무상 익금이지만 회계상 수익 X
  • 손금산입: 세무상 손금이지만 회계상 비용 X
  • 익금불산입: 회계상 수익이지만 세무상 익금 X
  • 손금불산입: 회계상 비용이지만 세무상 손금 X (접대비 한도초과·기부금 한도초과·법인세 등)
