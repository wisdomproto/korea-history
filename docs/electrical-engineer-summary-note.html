1. 전기자기학 — 정전기·자기

쿨롱의 법칙

F = (1/4πε₀) × (Q₁Q₂/r²) [N], k = 9×10⁹ N·m²/C²

전기장·전위

  • 전기장 E = F/Q [V/m]
  • 전위 V = W/Q [V] (전기장 내 단위전하 일)
  • 전기장과 전위 관계: E = -dV/dx
  • 가우스 법칙: ∮E·dS = Q/ε₀ (폐곡면 내 전하)

커패시턴스

  • C = Q/V [F]
  • 평행판: C = ε₀ε_r·A/d
  • 구형: C = 4πε₀·R
  • 직렬: 1/C = 1/C₁ + 1/C₂ + ... (저항 병렬과 같음)
  • 병렬: C = C₁ + C₂ + ...
  • 축적 에너지: W = (1/2)CV² = (1/2)QV = Q²/(2C)

자계·자기 인덕턴스

  • 비오·사바르: dB = (μ₀I/4π) × (dl×r̂/r²)
  • 앙페르: ∮H·dl = I (폐곡선 둘레 자계)
  • 패러데이: e = -N·dΦ/dt (전자유도 기전력)
  • 인덕턴스 L: V = L·di/dt
  • 축적 에너지: W = (1/2)LI²

전자력·로런츠

  • 전선 전자력: F = BIl·sinθ [N]
  • 로런츠 힘: F = qE + qv×B

3. 전기기기 — 변압기·전동기·발전기

변압기 (Transformer)

  • 권수비: a = N₁/N₂ = V₁/V₂ = I₂/I₁
  • 등가회로: 1차·2차 환산. 누설 리액턴스·여자전류
  • 손실: 무부하손 (철손, 히스테리시스 + 와류) + 부하손 (동손)
  • 효율: η = 출력/(출력+손실). 최대효율 = 철손 = 동손 시
  • 결선: Y-Y, Δ-Δ, Y-Δ, Δ-Y, V-V (3상 4종 + V결선)
  • 병렬운전 조건: 극성·권수비·임피던스·내부저항·% 임피던스 같아야

유도전동기

  • 슬립 s = (Ns - N)/Ns (동기속도 vs 회전속도)
  • Ns = 120f/P (동기속도, p=극수)
  • 토크: T = (P/4πf)·(s·R₂'/(R₂'² + (s·X₂')²))·... 비례추이
  • 기동법: 직입·Y-Δ·리액터·기동보상기·인버터
  • 속도제어: 슬립·극수·주파수 제어 (인버터 VVVF)

동기기 (발전기·전동기)

  • 동기속도 Ns = 120f/P
  • 동기 임피던스: 전기자 반작용 + 누설 리액턴스
  • 위상: 부하 시 진상·지상 (V곡선)
  • 난조: 부하 변동에 의한 진동. 제동권선으로 방지

직류기

  • 발전기 분류: 타려·자려 (분권·직권·복권)
  • 전동기 분류: 분권 (정속도)·직권 (가변)·복권
  • 전기자 반작용: 자속 감소·중성축 이동. 보상권선·보극으로 보정
📌 출제: 변압기 효율·유도전동기 슬립 매년.

4. 회로이론 — 직류·교류·과도

직류 회로

  • 옴 법칙: V = IR
  • 키르히호프 법칙: KCL (전류 합 0)·KVL (전압 합 0)
  • 저항 직병렬: 직렬 R = R₁+R₂..., 병렬 1/R = 1/R₁+1/R₂...
  • 전력: P = VI = I²R = V²/R [W]

교류 회로

  • 주기 T = 1/f, 각주파수 ω = 2πf
  • 실효값: 정현파 V_rms = Vm/√2
  • 임피던스: Z = R + jX. |Z| = √(R²+X²)
  • 유도성 X_L = ωL / 용량성 X_C = 1/(ωC)
  • 역률: cosθ = R/|Z|
  • 유효·무효·피상전력: P=VI·cosθ, Q=VI·sinθ, S=VI

공진

  • RLC 직렬 공진: ω₀L = 1/(ω₀C) → f₀ = 1/(2π√(LC))
  • 공진 시 |Z| = R (최소), I 최대
  • Q 인자: Q = ω₀L/R = 1/(ω₀CR). 선택도 (좁을수록 Q ↑)

과도현상

  • RC 회로 충전: V_C(t) = V(1 - e^(-t/RC))
  • RL 회로 전류: I(t) = (V/R)(1 - e^(-Rt/L))
  • 시정수: τ_RC = RC, τ_RL = L/R

3상 회로

  • Y결선: V_L = √3·V_P (선간/상). I_L = I_P
  • Δ결선: V_L = V_P. I_L = √3·I_P
  • 3상 전력: P = √3·V_L·I_L·cosθ = 3·V_P·I_P·cosθ
📌 출제: 임피던스·공진 주파수 매년.

  • G(s) = Y(s)/X(s) (출력/입력 비)
  • 1차 시스템: G(s) = K/(τs+1). 시정수 τ
  • 2차 시스템: G(s) = ωn²/(s² + 2ζωn·s + ωn²). 감쇠비 ζ

블록선도·신호흐름

  • 직렬: G₁G₂
  • 병렬: G₁+G₂
  • 피드백: G/(1±GH)
  • 메이슨 공식: 신호흐름선도

안정성·근궤적

  • 특성방정식 근: 좌반평면 → 안정 / 우반평면 → 불안정 / 허수축 → 임계
  • 루스-허위츠: 표 작성, 첫 열 부호 변화 = 우반평면 근 수
  • 나이퀴스트: 개루프 → 폐루프 안정성
  • 보드선도: 이득여유·위상여유

제어 종류

  • PID 제어: 비례+적분+미분
  • P: 빠른 응답, 정상편차 X 제거 / I: 정상편차 0 / D: 오버슈트 ↓·진동 ↓
  • 샘플 데이터 제어: 이산 시스템·z-변환

6. 전기설비 — 한국전기설비규정 (KEC)

저압·고압·특고압 (KEC)

  • 저압: 교류 1,000V 이하 / 직류 1,500V 이하
  • 고압: 1,000V 초과 ~ 7,000V (교류) / 1,500V 초과 ~ 7,000V (직류)
  • 특고압: 7,000V 초과

접지 (KEC 140)

  • 접지목적: 인체 보호·기기 보호·계통 운용
  • 저압 계통접지방식: TN(TN-S·TN-C·TN-C-S)·TT·IT
  • 접지저항 측정: 콜라우슈 브리지·어스 테스터

전선·케이블

  • 전선 굵기: 단면적 [mm²] 또는 AWG. 허용전류·전압강하 고려
  • 전선 식별: 상도체 (R/S/T 또는 L1/L2/L3)·중성선 (N, 청색)·보호도체 (PE, 녹색-노란색)
  • 절연저항: 메가옴미터 (메거)로 측정. 1MΩ 이상 (저압)

전압강하·허용전류

  • 전압강하: e = (2·I·L·R)/(1000·V) × 100 [%]
  • 표준 허용범위: 인입선 1%·간선 1%·분기 2%·총 5% 이내
  • 허용전류: 전선 굵기·온도·포설 방법별 표

배전·수변전 설비

  • 수전 방식: CB·LBS·ASS
  • 변압기 설치 방법: 옥내·옥외 (오일·몰드·건식·아몰파스)
  • 역률 개선: 콘덴서 (진상). 무효전력 보상
📌 출제: 저압·고압·특고압 구분 + 접지방식 매년.

7. 조명·전열·전동력 응용

조명

  • 광속 [lm]: 광원이 방출하는 빛의 양
  • 광도 [cd]: 단위 입체각당 광속
  • 조도 [lx]: 단위 면적당 광속 = lm/m²
  • 휘도 [cd/m²]: 광원 표면 밝기
  • 조명 계산: F = (E·A·D)/(N·U·M) [lm], D=감광보상률, U=조명률, M=보수율
  • 광원 종류: 백열·형광·HID(수은·메탈할라이드·고압나트륨)·LED

전열

  • 발열량 Q = I²R·t [J] (줄 법칙)
  • 1 cal = 4.186 J
  • 비열 c, 열용량 = m·c, 온도 변화 Q = mcΔT

전동력 응용

  • 펌프·송풍기: 동력 P = (Q·H·ρ·g)/η [W]
  • 승강기·엘리베이터: 균형추·로프·기어·인버터 제어
  • 전기철도: DC·AC·25kV·교류급전

축전지·UPS

  • 1차전지: 망간·알칼리·리튬 (1회용)
  • 2차전지: 납축·니카드·니켈수소·리튬이온·리튬폴리머
  • UPS: 무정전 전원 공급. 정류·인버터·축전지·바이패스
📌 출제: 조도·광속 + 전열 줄법칙.

9. 신재생·스마트그리드

태양광 (PV)

  • 구성: 태양전지 모듈 + 인버터 + 접속함 + 차단기 + ESS
  • 최대전력점추종 (MPPT): 출력 최대화 제어
  • 효율: 단결정 18~22% / 다결정 15~18% / 박막 10~15%

풍력

  • 출력: P = (1/2)·ρ·A·v³·Cp (베츠 한계 Cp ≤ 0.593)
  • 유형: 수평축 (HAWT)·수직축 (VAWT)
  • 육상풍력 vs 해상풍력

연료전지·수소

  • 종류: PEMFC (고분자전해질)·SOFC (고체산화물)·MCFC (용융탄산염)·PAFC (인산형)·AFC (알카라인)
  • 수소 생산: 그레이 (천연가스)·블루 (CCUS)·그린 (재생에너지 전기분해)

스마트그리드·ESS·V2G

  • 스마트그리드: 양방향 전력·정보 흐름. AMI (스마트미터)·SCADA·DR (수요반응)
  • ESS: 에너지저장시스템 (BESS·PHS·CAES·플라이휠·열저장)
  • V2G: Vehicle to Grid. 전기차 배터리 → 전력망
  • 마이크로그리드: 자급자족형 소규모 전력망
📌 출제: 태양광·풍력 출력 공식.
