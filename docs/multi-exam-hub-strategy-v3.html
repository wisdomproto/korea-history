Positioning Statement

한국에서 시험으로 신분이 바뀌는 30만 명(공무원·자격증·한능검 응시자)을 위해, 인강 ₩50~180만의 대안이나 보완재가 필요한 이들에게, 기출노트는 547개 시험 · 100만 문항 · 자동 매핑 요약노트를 제공하는 한국 시험 universe의 무료 학습 운영체제다. 에듀윌·해커스·메가공무원과 달리, 우리는 학원이 인수하거나 광고하고 싶은 보완재 위치를 차지한다.

3. 시즌 D-Day (Seasonal)

시험 4주 전 부터 매주 카운트다운 + 핵심 정리. 한능검 5/8/10월, 9급 4/6월, 자격증 시즌. 시즌별 8편.

4. 합격 인터뷰 (Social Proof)

실제 합격자 / 인증샷 / 학습 패턴 — 짧은 글 + 카드뉴스. K-factor 핵심 driver. 월 2편.

5. 시험 정책 뉴스 (Newsjacking)

"2027 한능검 인증제 정리" / "9급 시험 변경 사항" — 정책 변화 뉴스에 즉시 콘텐츠로 대응. 분기 2~3편.
