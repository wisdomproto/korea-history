1. 자산회계 (재고·유형·무형·금융자산)

TOPIC 0142회

재고자산 (K-IFRS 1002)

  • 취득원가: 매입가 + 매입부대비용 (운송·하역·관세·환급불가능 매입세금) - 매입할인·에누리·환출
  • 단가산정: 개별법·선입선출(FIFO)·가중평균법. 후입선출(LIFO) 금지 (K-IFRS 1002)
  • 저가법 (LCNRV): 취득원가 vs 순실현가능가치 중 낮은 금액. 평가손실 = 매출원가 가산
  • 차감 손실 환입 가능: 후속기에 NRV 회복 시 (단 원래 평가손실 한도)

유형자산 (K-IFRS 1016)

  • 인식조건: 미래경제적 효익 유입 가능 + 원가 신뢰성 측정
  • 최초측정: 원가 = 구입가격 + 직접관련원가 (운송·설치·전문가수수료) + 복구비용 추정치 - 매입할인
  • 후속측정: 원가모형 vs 재평가모형 (선택 가능, 자산 분류별)
  • 감가상각: 정액법·정률법·연수합계법·생산량비례법. 잔존가치·내용연수·방법은 매년 재검토
  • 손상 (K-IFRS 1036): 회수가능액 (사용가치·공정가치 - 처분비용 중 큰 금액) < 장부금액 시 손상차손

무형자산 (K-IFRS 1038)

  • 인식조건: 식별가능성·통제·미래경제효익
  • 내부창출 무형자산: 연구단계 비용 처리, 개발단계 — 6개 요건 충족 시 자본화 (기술적 실현가능성·완료 의도·사용·시장 존재·자원 충분·신뢰성 측정)
  • 영업권: 자체 창출 인식 X. 기업결합으로 취득한 영업권만 (상각 X, 매년 손상검사)
  • 내용연수 비한정 무형자산: 상각 X, 매년 손상검사

금융자산

  • 분류: ① 상각후원가측정 (AC), ② 기타포괄손익공정가치측정 (FVOCI), ③ 당기손익공정가치측정 (FVPL)
  • 분류 기준: 사업모형 + 계약상 현금흐름 특성
  • 지분상품: 원칙 FVPL. 단 단기매매 X면 FVOCI 선택 가능 (취소불가)

2. 부채·자본 (사채·충당부채·자본)

TOPIC 0235회

사채 (K-IFRS 1109)

  • 발행가: 미래현금흐름의 현재가치 (시장이자율 = 유효이자율 할인)
  • 액면발행·할인발행·할증발행: 표시이자율 vs 유효이자율 비교
  • 유효이자율법: 매기 사채이자비용 = 기초장부금액 × 유효이자율. 사채할인발행차금 상각액 = 이자비용 - 표시이자
  • 장부금액 변화: 할인발행 — 매기 증가 (액면금액 수렴) / 할증발행 — 매기 감소

충당부채 (K-IFRS 1037)

  • 인식조건: ① 과거사건의 결과로 현재의무 존재 ② 자원유출 가능성 높음 ③ 신뢰성 추정 가능
  • 측정: 의무이행에 필요한 지출의 최선의 추정치 (현재가치)
  • 우발부채: 자원유출 가능성 낮음 또는 신뢰성 추정 X — 인식 X (주석 공시)
  • 우발자산: 인식 X (실현 거의 확실 시만 자산 인식)

자본

  • 구성: 자본금·자본잉여금·자본조정·기타포괄손익누계액·이익잉여금
  • 주식배당: 자본 변동 X (이익잉여금 → 자본금 대체)
  • 현금배당: 자본 감소 (이익잉여금 ↓)
  • 자기주식: 자본조정 차감

5. 원가회계 (개별·종합·활동기준)

TOPIC 0520회

원가의 분류

  • 제조원가 vs 비제조원가: 제품원가 vs 기간비용
  • 직접원가 vs 간접원가: 추적가능성
  • 변동원가 vs 고정원가: 조업도 변동에 따른 변화
  • 관련원가 vs 매몰원가: 의사결정 관련성

개별원가계산 (Job-order Costing)

  • 고객 주문별 원가 집계 (주택건설·항공기·맞춤제품)
  • 실제원가계산 vs 정상원가계산: 정상원가 = 직접원가는 실제 + 제조간접원가는 예정배부
  • 예정배부율 = 예정 제조간접원가 ÷ 예정 배부기준 (직접노무시간·기계시간 등)
  • 배부차이: 실제 - 배부 → 비례배분/매출원가 가감

종합원가계산 (Process Costing)

  • 대량생산·표준화 (석유화학·화학·식품)
  • 완성품환산량: 재공품의 완성도 반영
  • 가중평균법 vs 선입선출법(FIFO): 기초재공품 처리 방식
  • 완성품원가 + 기말재공품원가 산출

활동기준원가계산 (ABC)

전통 원가계산이 다품종 소량생산에서 왜곡 → ABC는 활동(activity)을 원가대상으로. 원가동인(cost driver)에 따라 자원 → 활동 → 제품 배부.

6. 관리회계 (CVP·변동원가·손익분기점)

TOPIC 0617회

CVP 분석 (Cost-Volume-Profit)

  • 공헌이익 = 매출액 - 변동비. 공헌이익률 = 공헌이익 ÷ 매출액
  • 손익분기점 매출액 = 고정원가 ÷ 공헌이익률
  • 손익분기점 판매량 = 고정원가 ÷ 단위당 공헌이익
  • 목표이익 분석: 목표매출 = (고정원가 + 목표이익) ÷ 공헌이익률
  • 안전한계율: (실제매출 - 손익분기매출) ÷ 실제매출

재고자산 단가결정 (원가흐름가정)

  • 개별법: 실제 매입원가 추적 (보석·자동차 등 고가)
  • 선입선출법 (FIFO): 먼저 매입 → 먼저 판매. 인플레이션 시 매출원가 ↓ 이익 ↑ (세금 ↑)
  • 가중평균법 (이동·총평균): 평균단가로 산출. 안정적 표시
  • 후입선출법 (LIFO): K-IFRS 금지 (역사적 원가 왜곡)
  • 저가법: 취득원가 vs 순실현가능가치 중 낮은 것 (재고자산평가손실)

소매재고법 (Retail Inventory Method)

  • 의의: 백화점·마트 등 다품종 취급 사업의 재고자산 평가
  • 원리: 매가 기준으로 기록한 재고자산을 원가 기준으로 환산
    • 원가율 = (기초재고 원가 + 매입원가) / (기초재고 매가 + 매입매가 + 가격 인상 - 가격 인하)
    • 매가 기준 기말재고 × 원가율 = 원가 기준 기말재고
  • 변형: 평균법 소매재고법, 선입선출법 소매재고법, 저가법 소매재고법

매출가격환원법 (Gross Profit Method)

  • 의의: 매출총이익률을 이용해 매출원가 추정 → 기말재고 추정
  • 산식: 추정매출원가 = 매출액 × (1 - 매출총이익률)
  • 활용: 분기·월별 보고서 작성, 재해·도난으로 인한 재고손실 추정
  • 한계: 정식 재무보고에는 사용 불가 (추정)

원가-조업도-이익 분석 — 고저점법 (High-Low Method)

  • 의의: 혼합원가에서 변동비와 고정비 분리하는 가장 단순한 방법
  • 방법:
    1. 최고 조업도 + 최저 조업도 두 점만 사용
    2. 변동비율 = (최고 조업도 원가 - 최저 조업도 원가) ÷ (최고 조업도 - 최저 조업도)
    3. 고정비 = 총원가 - (변동비율 × 조업도)
  • 장점: 계산 간단
  • 단점: 두 점만 사용 → 비대표성, 회귀분석보다 부정확

원가배부 — 단계배부법 (Step Method)

방법특징
직접배부법보조부문 → 제조부문에만 직접 배부 (보조부문 간 무시)
단계배부법보조부문 간 일방적 배부 (배부순서 결정 후, 한 번 배부된 부문은 다시 배부 안 받음)
상호배부법보조부문 간 쌍방 배부 (연립방정식). 가장 정확
  • 단계배부법 배부순서 결정: ① 다른 보조부문에 더 많이 배부하는 부문 우선, ② 원가가 큰 부문 우선, ③ 용역제공 비율이 큰 부문 우선

종합원가계산

  • 대상: 동종·대량 생산 (시멘트·정유·화학·제분)
  • 완성품환산량: 진행중 작업을 완성품 단위로 환산
    • 평균법: 기초재공품 + 당기투입을 함께 처리
    • 선입선출법: 기초재공품과 당기투입 분리
  • 공손·감손:
    • 정상공손 (계획된 공손): 양품 원가에 배부
    • 비정상공손: 당기 손실 처리
  • 공정별 원가계산: 공정마다 원가 누적 → 다음 공정 전공정원가

관련 키워드 인덱스

종합원가계산방법 = 평균법·선입선출법 (위 종합원가계산 참조) / 매출가격환원법 = Gross Profit Method / 소매재고법 = Retail Inventory Method / 고저점법 / 단계배부법 / 계속기록법 / 실지재고조사법 / 기업회계기준 (K-IFRS·K-GAAP).

표준원가·차이분석

  • 표준원가 = 표준수량 × 표준가격
  • 직접재료비 차이:
    • 가격차이 = (실제가격 - 표준가격) × 실제수량
    • 능률차이 = (실제수량 - 표준수량) × 표준가격
  • 직접노무비 차이:
    • 임률차이 = (실제임률 - 표준임률) × 실제시간
    • 능률차이 = (실제시간 - 표준시간) × 표준임률
  • 변동제조간접비: 소비차이 + 능률차이
  • 고정제조간접비: 예산차이 + 조업도차이
