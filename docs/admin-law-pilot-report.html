미커버 2건 분석

❌ 2014-6 — 〈보기〉 형식 종합 문제

"〈보기 1〉의 내용을 근거로 판단할 때 〈보기 2〉 설명의 옳고 그름이 바르게 나열된 것은?" — 본문에 단원 키워드 X. 보기 내용을 보지 않으면 분류 불가. 모든 시험 노트의 한계 (학원 단권화도 동일). 사용자가 노트 + 풀이를 같이 보면 cover 가능.

⚠️ 2013-2 — 단원 누락

"행정주체가 될 수 없는 것은?" → '행정주체' 단원이 §17에 없음. 다음 버전에서 §17 행정법관계 단원에 "행정주체 (국가·지방자치단체·공공조합·영조물법인·사인의 행정사무 위탁)" 섹션 추가하면 cover 가능.

→ 보강 시 커버리지 97.5%로 상승 예상.

✅ 2. 한 문제 → 평균 2.8단원 매핑

law.go.kr 핵심 조문 + 대법원 판례 + 학설 = 1차 출처 기반 노트. AI hallucination 위험 X. "출처 footnote 명시" = 학원 단권화도 못 하는 신뢰성.
