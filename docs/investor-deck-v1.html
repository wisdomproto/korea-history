시장의 균열

시험은 1,300개인데, 학습 자료는 시험마다 따로 흩어져 있다.

01

자료가 시험마다 분산

같은 학습자가 한능검(60K 검색어) → 9급 → 자격증을 응시하는데, 각 시험마다 카페·블로그·PDF로 흩어진 자료를 새로 찾아야 함.

02

인강·학원은 비싸다

인강 패키지 ₩30~80만 / 학원 ₩100만+ / 요약 교재 ₩2~4만/권 — 공무원 수험생 평균 1년 학습비 ₩200만+ [업계 추정].

03

무료 사이트는 UI 구식 + 학습 도구 X

경쟁 무료 기출 사이트 (공기출·comcbt·맞추다) UI 노후 + 모바일 가독성 낮음. 오답노트·약점 분석·요약노트 매칭 모두 X — 학습자가 매번 수동 정리.

04

요약과 문제가 따로 논다

이론 정리 노트와 기출이 별개 자료 — "이 문제가 어떤 주제인가"를 학습자가 매번 수동 매핑. 같은 주제 다른 회차 묶기는 불가능.
