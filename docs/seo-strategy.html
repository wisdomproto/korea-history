02 3가지 핵심 인사이트

Insight 01 — 사이트가 "한국사 사전"으로 인식되기 시작

지금 들어오는 비브랜드 검색어는 한능검 시험 키워드가 아니라 학술/교과 롱테일 키워드:

혜공왕 업적 · 개화반대론자 · 검교태보의 직 · 좨주 직강 · 환국 순서 · 후삼국 통일 과정 · 1987년 6월 민주항쟁

→ 사용자는 "한능검 답"이 아니라 한국사 개념을 정리하려고 우리 사이트에 옴. 콘텐츠 전략 방향 = 역사 인물·사건·개념 위키 포지션 강화.

Insight 02 — 노트(Notes)가 SEO 주력, 시험(Exam)은 약함

168개 페이지가 노출됐는데 클릭이 발생한 페이지 분포:

· 홈 + /study + /notes/* = 92% 클릭 (45/49)

· /exam/* = 130개 노출되지만 클릭은 5건뿐, CTR 0~50% 양극화

이유: exam 페이지는 텍스트가 적어서 Google이 약한 매치만 함. 제목·선지·해설만 있고 주제 설명문 없음. → 노트 페이지에 SEO 자원 집중하는 게 효율적.

Insight 03 — 0% CTR 페이지 5곳, 메타 디스크립션 시급

노출은 많은데 클릭률 0%인 페이지는 제목/메타 매력도 문제:

· /study/custom (20 노출, 0 클릭, 평균순위 3.3) ← 1페이지인데 클릭 0

· /notes/s1-08 (11 노출, 0 클릭, 순위 7.6)

· /notes/s5-02 (10 노출, 0 클릭, 순위 11.9)

· /exam/76, /exam/63, /exam/49 (각 8~9 노출, 0 클릭)

→ 제목/메타 디스크립션 개선만으로 CTR 5% 끌어올려도 클릭 +수십.

03 브랜드 & 헤드텀 SEO ⭐ 최우선

롱테일 콘텐츠 전략은 시간이 걸리지만, 브랜드 & 카테고리 헤드텀은 사이트의 정체성입니다. 지금 위험 신호가 있어 가장 먼저 점검 필요.

🚨 위험 신호 — "기출노트" Google 미인식

90일 GSC 데이터에 기출노트 검색이 0건. gcnote로는 1위(9 클릭)지만, 한국어 브랜드명 "기출노트"가 우리 사이트와 매치되지 않는 상태.

의미: 누군가 "기출노트"로 검색해도 우리 사이트 노출 안 됨 → 브랜드 자산이 검색 엔진에 적립되고 있지 않음. 유튜브/SNS/광고로 브랜드 알려도 → 검색으로 찾으면 다른 사이트가 노출되는 위험.

  • 홈페이지 H1·메타 강화: 첫 줄에 "기출노트" 명시. 현재 메타 타이틀이 "한능검"만 강조하면 → "기출노트 한능검 — ..." 형식
  • 모든 페이지 사이트명 통일: <title>의 사이트 suffix를 "| 기출노트" 또는 "| 기출노트 한능검"으로 (현재 "| 한능검"만 있다면)

  • Footer 브랜드 강화: 푸터/About 영역에 "기출노트는 ..."로 시작하는 사이트 설명 1단락 추가
  • 사이트 가치제안 푸터 (카드뉴스 v1과 통일): 모든 노트/문제 페이지 하단에 "기출노트 — 한국사능력검정시험 무료 학습 플랫폼" 마이크로카피
  • 외부 링크 자연 인용 유도: 카드뉴스/스레드/블로그 운영 시 "기출노트(gcnote.co.kr)"로 일관 표기. 외부 백링크의 anchor text가 브랜드명 시그널이 됨

  • /notes 인덱스 — "한능검 요약노트" 정조준: H1을 "한능검 요약노트 — 시대별 87편 무료"로. "요약노트"라는 단어를 본문 자연 노출 6~10회. 이미 "한능검 요약" 4위이므로 한 발 더 밀면 1~2위 가능
  • /exam 인덱스 — "한능검 기출문제" 정조준: H1 "한능검 기출문제 — 40~77회 1,900문제 무료". 회차 카드 그리드. 본문에 시험 소개 + 합격선 1단락
  • 홈 — "기출노트 한능검" 정조준: Hero H1을 "기출노트 — 한능검 기출문제와 요약노트"처럼 헤드텀 4개를 한 문장에 자연스럽게. 메타 디스크립션도 동일

전략 원칙 — 브랜드 ⊃ 헤드텀 ⊃ 롱테일

SEO 자산은 3층 구조로 쌓임: ① 브랜드 (기출노트) 가 정체성 — 1주 내 정립 / ② 헤드텀 (한능검·요약노트·기출문제) 이 카테고리 — 1개월 내 진입 / ③ 롱테일 (인물·사건·개념) 이 트래픽 폭 — 3개월 내 확장. 위에서 아래로 내려가는 게 정공법.

04 Quick Win 7개 — 30일 내 실행

기존 페이지의 메타/콘텐츠만 손봐도 +50~100% 클릭 증가 가능. 신규 콘텐츠 없이 즉시 실행.

05 황금키워드 패턴 — 롱테일 발굴 공식

90일 데이터에서 발견한 실제 검색 패턴을 일반화. 이 공식으로 신규 콘텐츠 우선순위 결정.

패턴 A — 인물 + 의도

{인물} + {업적·평가·정책·정리·생애}

  • 혜공왕 업적 [실측]
  • 정조 업적 / 광해군 업적
  • 흥선대원군 정책
  • 이성계 / 정도전 / 사도세자

패턴 B — 사건 + 정리

{사건} + {과정·결과·요약·정리·원인·배경·의의}

  • 후삼국 통일 과정 정리 [실측]
  • 1987년 6월 민주항쟁 [실측]
  • 갑신정변 결과 / 임진왜란 과정
  • 을사조약 내용 / 동학농민운동 결과

패턴 C — 제도/개념 + 설명

{제도/개념} + {내용·뜻·차이·종류}

  • 검교태보의 직 [실측]
  • 좨주 직강 [실측]
  • 환국 순서 / 환국 정리 [실측]
  • 음서제 / 노비안검법 / 사림과 훈구

패턴 D — 시험 직접 검색

한능검 + {회차·답지·요약·범위}

  • 한능검 요약 [실측, 4위] ⭐
  • 한능검 77회 답지 [실측]
  • 한능검 / 한국사능력검정 [실측]
  • 한능검 1급 합격선 / 출제경향
행동 원칙

패턴 A·B·C가 우리 강점 — 노트 페이지가 이미 잘 매치되고 있음. 시대별 노트에 인물/사건 별 anchor 섹션을 명확히 추가하면 자동으로 수백 키워드 노출 잠재.

07 약한 페이지 진단 — exam 페이지 SEO 보강

exam 페이지 130개가 GSC에 노출되지만 클릭은 미미. 1,900+ 문제가 SEO 자산이 되려면 보강 필요.

현재 문제

  • 텍스트 부족: 문제 + 선지 + 짧은 해설만. Google이 매치할 본문이 적음
  • 중복 패턴: 모든 페이지가 비슷한 구조 → 차별화된 키워드 시그널 약함
  • 롱테일 매칭 약함: "검교태보의 직" 같은 마이너 키워드가 노트가 아닌 exam 페이지로 라우팅됨 (잘못된 매칭)

개선안

  • 문제별 "핵심 키워드 + 시대 + 유형" 섹션 추가 — 정답 위에 1줄로: "이 문제는 [고려 후기 - 정치] 유형 / 핵심 인물·사건"
  • "관련 노트" 링크 강화 — 이미 구현된 era→note 매핑을 SEO 친화 anchor 텍스트로 (예: "이 문제와 관련된 통일신라 정치 요약 보기")
  • 회차 페이지 (/exam/77)에 시험 메타 정보 추가: 시험 일정, 출제 경향 한 단락, 합격선 — long content for ranking
  • FAQ 스키마: 회차 페이지에 "이번 회 합격선?", "난이도?", "출제 비중?" 3~5개 Q&A로 리치 결과 노릴 가치
