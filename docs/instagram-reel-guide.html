촬영 시나리오

3개 클립으로 나눠 찍고 인스타에서 합치세요

⏱️ 녹화 시작 후 3초 기다렸다가 조작 시작, 마지막 동작 후 2초 유지 (앞뒤 자르기 여유)
1
메인 → 문제 풀기
약 10초 · 정답 효과음이 하이라이트
0 - 2초
메인 페이지 보여주기
살짝 아래로 스크롤 (퀵액션, 최신기출 영역)
2 - 4초
"최신 기출"에서 77회 터치
또는 상단 "기출문제" 메뉴 → 회차 선택
4 - 6초
문제 목록 → 1번 문제 터치
문제 화면 진입, 지문 이미지 잠깐 보여주기
6 - 8초
살짝 스크롤해서 선지 노출
이미지 + 선지가 함께 보이게
8 - 10초
정답 선지 터치!
confetti + 스파클 + 점수 팝업
🎉 효과음이 들려야 합니다 — 볼륨 확인!
2
해설 + 유튜브 + 요약노트
약 10초 · 컨텐츠 깊이를 보여주는 구간
0 - 3초
정답 맞춘 직후 아래로 스크롤
AI 해설 텍스트 노출
3 - 5초
더 스크롤 → 유튜브 영상 노출
영상 썸네일 보이면 OK (재생 안 해도 됨)
5 - 7초
"관련 요약노트" 버튼 터치
초록색 버튼 — 스크롤 올려서 찾기
🔗 노트 페이지로 자동 이동
7 - 10초
요약노트 내용 천천히 스크롤
텍스트가 잘 보이게 느린 속도로
3
오답노트
약 5~8초 · 복습 기능 어필
0 - 2초
헤더에서 "오답노트" 이동
하단 메뉴 또는 상단 네비게이션
2 - 5초
틀린 문제 리스트 보여주기
날짜/시간 표시되는 거 보이게
5 - 8초
아무 문제 터치 → 다시 풀기
"복습" 느낌 전달

촬영 꿀팁

퀄리티를 높이는 디테일

⚡
속도감 유지
각 동작 사이 1초 이상 멈추지 마세요. 릴스는 2초 이상 같은 화면이면 이탈률 급상승합니다.
👆
손가락 위치
화면 하단 1/3에서 터치. 손가락이 컨텐츠를 가리지 않게. 스크롤은 오른쪽 가장자리에서.
🎬
여유 있게 녹화
녹화 시작 후 3초 대기, 마지막 동작 후 2초 유지. 앞뒤를 나중에 깔끔하게 자를 수 있습니다.
🔄
여러 번 찍기
각 클립 2~3번씩 찍어보고 제일 나은 거 쓰세요. 첫 릴스니까 완벽하지 않아도 괜찮아요!
📱 화면 녹화 방법 — iOS: 제어센터 → 화면 기록 (3초 카운트다운) / Android: 퀵패널 → 화면 녹화
4

인스타 편집

촬영한 클립을 릴스로 완성하기

1
클립 합치기
인스타 → 릴스 → "+" → 갤러리에서 클립 3개 순서대로 선택. 각 클립 앞뒤 불필요한 부분 트리밍.
2
텍스트 오버레이 추가
각 구간에 맞춰 아래 텍스트를 올려주세요. 굵은 폰트, 흰색 + 검정 그림자, 화면 상단 1/4 위치에.

텍스트 오버레이 타이밍

0초한능검 이렇게 공부하면 됨 👇
4초기출 풀고
8초맞추면 이 효과 ㄷㄷ
12초AI가 해설해줌
15초유튜브 해설 영상까지
18초요약노트로 정리
22초틀린 것만 다시 복습
25초무료임 ㄹㅇ → gcnote.co.kr
⚠️ 텍스트는 화면 상단 1/4에 배치하세요. 하단은 인스타 UI(좋아요, 댓글, 공유 버튼)에 가려집니다.
3
BGM 추가
인스타 음악 검색 → "study", "motivation" 계열 또는 트렌딩 음악 (도달률 높음). 볼륨은 낮게 — 효과음이 들려야 하니까.
4
커버 이미지 선택
정답 맞추는 순간 (confetti 터지는 장면) 캡처가 제일 눈길 끕니다.
