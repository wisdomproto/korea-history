2. 어법 (시제·태·관계대명사·가정법) 출제 28회

시제 (Tense) — 12시제 핵심

시제형태용법
현재V/Vs현재 사실, 일반 진리, 습관
과거V-ed과거 사실, 역사적 사실

현재완료have/has p.p.경험·계속·완료·결과 (현재까지)

현재진행am/is/are -ing지금 진행
과거진행was/were -ing과거 진행
  • 시제 일치: 주절 과거 → 종속절 과거/과거완료 (단 진리·습관·역사적 사실은 예외)
  • 시간·조건의 부사절: 미래시제 대신 현재시제 (when, if, before, after, until, as soon as)

태 (Voice) — 능동·수동

  • 수동태 형태: be + p.p. (+ by + 행위자)
  • 완료 수동태: have/has been p.p.
  • 진행 수동태: be being p.p.
  • 주의 자동사: 수동태 불가 (happen, occur, appear, arise, exist, belong, consist)
  • 4형식 수동: 직접목적어·간접목적어 둘 다 주어 가능 (단 buy·make 등은 직접목적어만)

관계대명사·관계부사

선행사주격목적격소유격
사람whowhomwhose
사물·동물whichwhichwhose/of which
사람·사물thatthat—
선행사 포함what (the thing which)
  • 관계부사: when(time), where(place), why(reason), how(way)
  • that 사용: 최상급, 서수, the only/the very/the same, all/no/every 뒤
  • 계속적 용법: , which / , who (앞 절 전체 또는 한 부분 부연). that은 계속적 용법 불가

가정법

유형if 절주절의미
가정법 과거If S + 과거동사 (was→were)S + would/could/might + V현재 사실 반대

  • I wish + 가정법 과거: 현재 이루지 못한 소망
  • as if/as though + 가정법 과거: 마치 ~인 것처럼 (사실은 아님)
  • If 생략 시 도치: Were I rich = If I were rich, Had I known = If I had known

분사·동명사·부정사

  • 분사구문: 부사절 → 분사 (Walking down the street, I saw...) — 능동/완료/수동 구분
  • 분사 의미상의 주어: 주절 주어와 다르면 명시 (The weather being fine, ...)
  • to부정사 vs 동명사:
    • to부정사만 목적어: want, hope, decide, plan, agree, refuse, promise, expect, manage
    • 동명사만 목적어: enjoy, finish, mind, avoid, give up, postpone, suggest, consider, deny, admit
    • 둘 다, 의미 다름: remember/forget/regret + to V (앞으로) vs -ing (이미)

기타 빈출 어법

  • 병렬 구조: and/but/or 양쪽 같은 형태 (V-ing and V-ing, to V and to V)
  • 주어-동사 수일치: 주어가 단수/복수 정확히 파악 (긴 수식어 주의)
  • 어순: 부정어구 도치 (Hardly/Scarcely/Never/Rarely + 조동사 + S + V)
  • 비교: as 원급 as / -er than / the -est / 비교 대상 형태 일치

3. 독해 — 주제·요지·제목 출제 24회

출제 패턴

  • What is the main idea/topic of the passage?
  • Which of the following is the best title?
  • The author's main purpose is to...

주제·요지 찾는 4가지 단서

  1. 첫 문장 (topic sentence): 영어 글은 두괄식이 80% 이상
  2. 핵심 명사 반복: 글에서 가장 자주 등장하는 개념
  3. 역접 후 진술: but, however, yet 다음에 핵심 주장
  4. 마지막 문장 결론: in conclusion, therefore, thus

제목 (title) 고르기

  • 주제보다 함축적·압축적
  • 비유·은유 활용 가능
  • 너무 좁거나 (사례 한정) 너무 넓은 (전체 학문) 선지는 오답

4. 독해 — 세부정보·일치·불일치 출제 21회

출제 패턴

  • Which of the following is true/NOT true according to the passage?
  • What does the passage say about ~?

풀이 전략

  1. 선지 먼저 읽기 — 키워드 메모
  2. 본문에서 키워드 위치 빠르게 검색 (skimming)
  3. 해당 부분 정독해 일치 여부 확인
  4. 추론 절대 금지 — 본문에 명시된 내용만

함정 5종 (한국어 시험과 동일)

  1. 부분 사실 + 일부 왜곡
  2. 정도 과장 (always, all, every, never, none)
  3. 인과 역전
  4. 본문에 없는 정보 추가
  5. 비교 대상 바꾸기

6. 독해 — 흐름·순서·삽입·빈칸 출제 11회

글의 순서 배열

  • 주어진 글 → (A)·(B)·(C) 배열
  • 전략: 지시어(this, that, these, those, it), 접속어(however, therefore, in addition, for example) 추적
  • 대명사가 지칭하는 명사가 직전에 와야 함

주어진 문장 삽입

  • 삽입할 문장 분석 → 무엇을 가리키는가? 어떤 흐름인가?
  • 삽입 후보 위치마다 앞뒤 흐름 확인 → 가장 자연스러운 곳

빈칸 추론

  • 주제 빈칸: 글 전체 요지 = 빈칸
  • 예시 앞 빈칸: For example 다음에 나오는 사례 → 빈칸은 일반 진술
  • 대조 후 빈칸: However 다음 빈칸 → 앞 내용과 반대

흐름과 무관한 문장 찾기

  • 주제와 동떨어진 문장 1개 색출
  • 같은 단어가 나와도 주제 흐름과 무관하면 정답

능동태·수동태 변환

  • 능동태 (Active Voice): S + V + O — 주어가 행위 (Tom wrote a letter)
  • 수동태 (Passive Voice): S + be + p.p. (+ by + 행위자) — 목적어가 주어로 (A letter was written by Tom)
  • 변환 절차:
    1. 능동태 목적어 → 수동태 주어
    2. 능동태 동사 → be + p.p. (시제 유지)
    3. 능동태 주어 → by + 행위자 (생략 가능)
  • 수동태 불가 동사: 자동사 (happen, occur, appear, arise, exist, belong, consist, resemble, lack)
  • by 외 전치사 사용:
    • be interested in / be surprised at / be filled with / be made of (재료) / be made from (가공)
    • be known to (사람) / for (특징) / by (수단) / as (자격)
