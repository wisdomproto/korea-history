한 줄 결론

"공기출은 콘텐츠량·트래픽 우위지만 자동 매핑·요약노트·모던 UI 없음. 우리는 시작 4주차로 콘텐츠량 따라잡지 못해도 UX × moat (문제↔노트 자동 매핑) × 다중시험 통합으로 차별화 가능. 12~24개월 SEO + SNS 자생 트래픽 + 광고 ₩2,000만/월 운영하면 공기출과 양강 구도 형성 가능."

02 — Tier 1 (직접 경쟁)공기출 Deep Dive

공기출 위협 ★★★★★

0gichul.com

"공무원기출, 문제해설, 본문검색" — 한국 무료 기출 시장 1위.

✅ 강점
  • 가장 큰 콘텐츠 — 2000년부터 데이터, 200+ 과목 분류
  • 공무원 5급~9급, 지방직, 경찰, 소방, 군무원, 한능검, PSAT, 수능까지 통합
  • 전문직 (감정평가사·관세사·노무사·법무사·변리사·세무사) 망라
  • 다크모드, 캘린더, 알림 기능
  • 활발한 커뮤니티 (질문/합격수기/장터)
  • AdSense + 카카오 듀얼 광고 운영 ✓ (우리와 동일 전략)
  • AI 기반 자동 해설 생성
  • 본문 검색 기능 (SEO + 사용자 가치)
❌ 약점
  • UI/UX 노후 (2010년대 디자인) — 모바일 first 아님
  • 요약노트 X — 기출 풀이만
  • 문제 ↔ 노트 자동 매핑 X (우리 진짜 moat)
  • 강의 영상 매칭 X
  • 오답노트 자동 수집 X
  • 개인 약점 분석 X
  • 광고 도배 (사이드/상단/사이사이) — 학습 흐름 방해
  • 모던 인터랙션 X (애니메이션/사운드/공유)
우리 vs 공기출

콘텐츠량: 공기출 2000년+ 200과목 vs 우리 한능검 1,900문제 + 547시험 + 100만 R2 문제. 우리 R2 데이터 절대량은 더 큼.
UI/UX: 우리 압도 (2026 모던 디자인 vs 2010년대).
학습 도구: 우리 압도 (요약노트 87개 + 자동 매핑 + 오답노트 + 개인 기록).
커뮤니티: 공기출 압도 (활발한 게시판). 우리 게시판 시드 단계.
SEO 권위: 공기출 압도 (도메인 연령 10+년 vs 우리 1개월).
판단: 우리는 UX × moat × 다중시험 통합으로 차별화. 12~24개월 SEO + SNS로 트래픽 갭 좁힘. 공기출의 핵심 약점 = 모바일 + UX + 자동 매핑 → 우리가 모두 이김.

03 — Tier 2 (카테고리 강자)comcbt · 맞추다

comcbt.com

"500개+ 자격증 완전 무료 CBT" — 자격증 카테고리 절대 강자.

  • 500개+ 자격증 망라 (기사·산업기사·기능사·컴활·정보처리·전기·소방·기계 등)
  • "지구인 누구에게나 무료" 표방 — 강한 무료 정체성
  • 오답노트 + 자동채점 + 모의고사 + 성적관리 (교사용/학생용)
  • 모바일 전용 사이트 (m.comcbt.com) + Android 앱
  • 도메인 연령 길어 SEO 권위 강함
❌ 약점
  • UI/UX 매우 낙후 (90년대~2000년대 디자인)
  • 요약노트 X (기출 풀이만)
  • 광고 운영 정보 약함 (학원 추천 섹션 위주)

  • 공무원·한능검 카테고리 약함
우리 vs comcbt

현재 겹침 작음 — 우리는 한능검 + 공무원 위주, comcbt는 자격증 위주. 단 우리 자격증 카테고리 본격 진출 (Phase 3 / Q3 후) 시 충돌. 그때 우리 차별점: 요약노트 + 자동 매핑 + 모던 UI로 "기출만" 사이트와 차별. comcbt는 SEO 권위 + 콘텐츠 망라 압도 → 정면 경쟁 X, "학습 도구가 있는 자격증 사이트" 포지션으로 차별화.

machuda.kr

"AI 기반 맞춤학습 + 프리미엄 구독" — 모던 UI + freemium 모델.

✅ 강점
  • 모던 UI/UX (우리와 비슷한 수준)
  • 전문가 해설 + 자동 채점 + 오답노트 자동

  • 도메인 연령 짧음 (우리와 비슷)

우리 vs 맞추다

06 — Position Map2×2 포지셔닝 맵

두 축으로 시장 그리면 우리 포지션이 보임. X축 = 학습 도구 깊이 (단순 풀이 ↔ 자동 매핑·노트·기록) / Y축 = UI/UX 세대 (레거시 ↔ 모던).

읽는 법

"모던 UI + 깊은 학습도구" 사분면(우상단)에 우리만 거의 단독. 맞추다는 가장 가까운 신생이지만 freemium 모델 + 자격증 위주 → 한능검·공무원 시장에선 우리만 차지. 이 사분면이 비어있는 이유는 (1) 콘텐츠 + UI 둘 다 만들기 어려움 (2) 무료 + 깊은 도구 = 수익화 모델 명확 X. 우리가 풀어낸 답: "학원 affiliate + 광고 + (Phase 4) 멤버십" 다각화.

07 — Moat우리 진짜 Moat (5개)

경쟁사 분석으로 명확해진 5개 차별점. 콘텐츠량 + SEO 권위는 시간이 해결 (12~24개월). 단, 아래 5개는 우리가 코드 + 디자인 + 데이터 인프라로 만든 구조적 우위 — 따라잡기 어려움.

1. ⭐ 문제 ↔ 노트 자동 매핑

정답 확인 순간 "이 문제는 노트의 X 부분과 관련" 자동 표시 + 1클릭 이동. 9개 경쟁사 중 0곳 보유. 우리 콘텐츠 파이프라인에 강제 룰로 박혀 있음 — 신규 시험 추가해도 자동 매핑.

2. 📚 요약노트 87개 + 강의 영상 매칭

한능검 시대별 87개 노트 (203,922자) + 최태성 강의 영상 124개 매칭. 공기출·comcbt 0개 / 학원사는 강의 위주, 텍스트 노트 X. 콘텐츠 깊이 차별화.

3. 🎨 2026 모던 UI/UX

cream + ink + amber + teal 디자인 시스템 / Noto Serif KR + Pretendard / 모바일 first 반응형 / 사운드 효과 + 애니메이션. 9개 경쟁사 중 맞추다만 비슷한 수준 (그것도 freemium).

4. 🌐 다중시험 통합 (547 ExamType)

한능검 + 9급 (24직렬) + 자격증 728개 R2 통합. 사용자 인생 시험 30년 동반 (운전→자격증→공무원→직무자격증). 공기출 (200과목) 외엔 단일 카테고리만.

5. 🔧 개인 학습 도구 풀세트

오답노트 자동 수집 / 내 기록 (점수·급수·약점 분석) / 학습 세션 (맞춤형) / 키워드별 학습. localStorage 기반 즉시 작동. 공기출·comcbt는 단편적 / 맞추다는 freemium에서만 풀세트.

✅ Moat 정리

5개 모두 코드/디자인/데이터 인프라 기반 구조적 우위. 경쟁사가 1년 안에 따라잡기 어려움. 우리는 그 사이 SEO + SNS + 광고로 트래픽 갭 좁힘.

08 — Action위협·기회 + 액션 함의

위협 (Threats)

🔴 공기출 — SEO 갭 너무 큼

공기출 도메인 연령 10+년 + 200과목 백링크 누적. 구글 검색 "9급 공무원 기출"에서 우리가 1순위 진입까지 12~24개월 필요.

대응: SEO 정면 경쟁 X → 모바일 + SNS (인스타·스레드) + longtail 키워드 (직렬별 합격 가이드)로 우회. 공기출이 약한 채널 점유.

🔴 NextStudy 재정비 후 (~2026 Q4)

매각 후 마케팅 공백 6~12개월. Q4 이후 재정비 완료 시 광고 + SNS 재가동 → 우리 SOV 줄어듦.

대응: Q3까지 SOV 50%+ 확보 마지노선. 9급 24직렬 노트 + 인스타·스레드 일일 발행으로 골든 윈도우 활용.

🔴 comcbt 자격증 SEO 압도

우리 자격증 진출 (Phase 3 / Q3 후) 시 comcbt와 충돌. comcbt는 도메인 연령 + 500개 자격증 망라 우위.

대응: 자격증 정면 경쟁 X → "학습 도구가 있는 자격증 사이트"로 차별. 변리사·노무사·감정평가사 등 고가치 자격증부터 진출.

🔴 맞추다 추격 (가장 비슷한 신생)

맞추다는 우리와 가장 비슷한 모던 사이트 + 자격증 강함. 향후 한능검·공무원 진출 시 직접 경쟁.

대응: "완전 무료 + 자동 매핑" 차별화. 맞추다는 freemium 핵심 기능 유료화 — 우리는 모든 기능 무료 (광고만).

기회 (Opportunities)

🟢 모든 경쟁사 자동 매핑 X

9개 사이트 중 0곳이 문제↔노트 매핑 보유. 우리 진짜 moat. 마케팅 메시지 1순위로 강조 — "정답 확인 순간 노트로 1클릭 이동, 다른 사이트엔 없는 학습 경험".

🟢 한국 EdTech SNS 부재

맞추다 외엔 SNS 활동 거의 X (공기출도 X). 인스타·스레드 = 빈 공간. 우리가 점유하면 SOV 독점 가능.

🟢 공기출 UI 갭 = 우리 무기

공기출 사용자 중 모바일 UX 불만 사용자 → 우리로 이동 가능. "공기출은 PC, 우리는 모바일" 마케팅 메시지.

🟢 NextStudy 매각 = 광고 채널 비음

메가공무원이 빠진 자리 = Naver/Google 광고 경쟁 약화. 우리 광고 ROAS ↑. 광고비 max 사용 (₩2,000만/월) 시 효율 극대화.

액션 함의 — Q2 OKR 우선순위

① 마케팅 메시지 1순위 = "자동 매핑" — 인스타 카드뉴스/스레드/SEO 모두 이걸로 통일. "다른 사이트엔 없는 학습 경험".
② 모바일 first 강조 — "공기출보다 모바일에서 빠르고 깔끔" 비교 콘텐츠 (직접 사이트명 X, 정성 비교)
③ 인스타·스레드 SOV 점유 — 한국 EdTech 빈 공간. 일일 발행으로 6개월 안에 카테고리 1위.
④ Q3 9급 24직렬 노트 입력 — 콘텐츠 깊이 차별화. 공기출 따라잡기 X, 직렬별 깊이 차별.
⑤ NextStudy BD 시드 컨택 (Q3 PV 100K 도달 후) — 매각 직후 lead 절실 = 답신 확률 가장 높음.

최종 한 줄 결론

"한국 무료 기출 시장 = 공기출 1위 + 카테고리 강자들. 우리는 '모던 UI + 자동 매핑 + 다중시험 통합' 사분면을 거의 단독 점유."

콘텐츠량 + SEO는 12~24개월에 따라잡기 어려움 → 정면 경쟁 X. 대신 UX × moat × SNS 빈 공간 점유로 차별화. 12~24개월 후 공기출과 양강 구도 형성 + 자격증 카테고리 진출로 추가 시장 흡수 가능.
