# PRD: 한국사능력검정시험 학습 앱 "한국사 마스터"

---

## 1. Summary

한국사능력검정시험(한능검) 응시자를 위한 모바일 학습 앱이다. 기출문제 CBT 모의고사, AI 기반 오답 분석, 맞춤형 학습 로드맵을 핵심으로 한다. 2027년 9급 공무원 시험 한국사 과목의 한능검 대체 시행으로 급증하는 응시 수요를 겨냥하여, 기존 앱 대비 "개인 맞춤 분석"과 "AI 해설"로 차별화한다.

## 3. Background

### Context

한국사능력검정시험은 국사편찬위원회가 주관하는 국가 공인 시험이다. 공무원 시험, 군무원, 교원 임용, 각종 공기업 채용에서 가산점 또는 필수 자격으로 활용된다.

### Why Now?

**2027년 9급 공무원 한국사 과목 폐지 및 한능검 대체**가 핵심 시장 촉매이다.

- 2027년부터 9급 공무원 시험에서 한국사 과목이 폐지되고, 한능검 3급 이상 취득으로 대체된다
- 이미 5급(2012년), 7급(2021년) 공채에서 한능검이 도입되어 있으며, 9급까지 확대 적용된다
- 2026년에는 응시자 수 증가에 대응하여 1회 증회 편성이 예정되어 있다
- 2021년 기준 연간 응시자 수가 51만 8천 명을 넘었으며, 9급 대체 시행으로 추가 증가가 예상된다

### What Recently Became Possible?

- **AI 기술 성숙**: LLM(Claude, GPT 등)의 한국어 역사 지식 정확도가 상업적 활용 수준에 도달
- **모바일 학습 습관 보편화**: 이동 중 학습, 짧은 시간 반복 학습 패턴이 표준이 됨
- **기존 앱의 정체**: 주요 경쟁 앱들이 단순 기출문제 제공에 머물러 있어 차별화 기회 존재

## 4. Objective

### 목표

한국사능력검정시험 응시자에게 **가장 효율적인 합격 경로**를 제공하는 모바일 학습 앱을 만든다.

### Why It Matters

- **사용자**: 시간 낭비 없이 취약 영역을 집중 공략하여 합격/목표 급수 달성
- **비즈니스**: 연간 50만+ 응시자 시장에서 유의미한 점유율 확보

### Key Results (SMART OKR)

| KR | 지표 | 목표 | 기간 |
|----|------|------|------|
| KR1 | 앱 다운로드 수 | 출시 후 3개월 내 10,000건 | Q3 2026 |
| KR2 | MAU (월간 활성 사용자) | 출시 6개월 내 5,000명 | Q4 2026 |
| KR3 | DAU/MAU 비율 (Stickiness) | 30% 이상 | Q4 2026 |
| KR4 | 앱스토어 평점 | 4.5 이상 유지 | 지속 |
| KR5 | 유료 전환율 | 무료 사용자 대비 5% | Q1 2027 |
| KR6 | 사용자 합격률 | 앱 사용자의 시험 합격률 80% 이상 (자기 보고) | Q1 2027 |

## 5. Market Segment(s)

### Primary Segment: 공무원/공기업 취업 준비생

- **문제**: 한능검을 빠르게 취득해야 하지만 한국사 학습에 과도한 시간을 쓰고 싶지 않음
- **동기**: 2027년 9급 한능검 대체로 인해 필수 자격 취득 필요
- **특성**: 시간 효율을 최우선 가치로 여김, 유료 결제 의향 상대적으로 높음
- **규모**: 연간 약 20~30만 명 (9급 공무원 응시자 기준)

### Secondary Segment: 학생 (중고등, 대학생)

- **문제**: 수행평가, 대학 입시 가산점, 취업 스펙을 위해 한능검 필요
- **동기**: 교양 증진 + 스펙 축적
- **특성**: 가격 민감도 높음, 게이미피케이션에 반응 좋음
- **규모**: 연간 약 15~20만 명

### Tertiary Segment: 일반 성인 학습자

- **문제**: 교양으로 한국사를 배우고 싶지만 체계적인 학습 도구가 부족
- **동기**: 자기계발, 역사 관심
- **특성**: 학습 지속률이 낮을 수 있어 동기 부여 장치 필요
- **규모**: 연간 약 5~10만 명

### 시장 제약 조건

- 시험이 연 4~6회만 시행되어 **수요의 계절성**이 뚜렷함
- 응시 후 합격하면 이탈 → **사용자 생애 가치(LTV)가 짧을 수 있음**
- 무료 기출문제가 국사편찬위원회 홈페이지에서 제공됨 → **무료 대안 존재**

## 7. Solution

### 7.1 UX / User Flow

#### 핵심 사용자 플로우

```
[앱 설치] → [온보딩] → [목표 설정] → [학습] → [분석] → [반복]

상세 플로우:

1. 온보딩 (최초 1회)
   ├── 목표 급수 선택 (심화 1~3급 / 기본 4~6급)
   ├── 시험일 설정 (D-day)
   ├── 하루 학습 시간 설정 (10분 / 20분 / 30분)
   └── 현재 수준 진단 테스트 (20문항, 5분)

2. 메인 홈
   ├── 오늘의 학습 카드 (추천 문제 + 복습 문제)
   ├── 학습 진행률 바
   ├── D-day 카운트다운
   └── 연속 학습일 (스트릭)

3. 학습 모드
   ├── [CBT 모의고사] 실전과 동일한 환경
   ├── [단원별 학습] 시대/주제별 문제 풀이
   ├── [오답 복습] 틀린 문제 재도전
   └── [일일 퀴즈] 5~10문제 빠른 풀이

4. 결과 & 분석
   ├── 점수 + 합격 예측
   ├── 시대별/유형별 정답률 차트
   ├── 취약 영역 Top 3
   └── "다음 학습 추천" 자동 생성

5. AI 해설 (문제별)
   ├── 정답 해설 (배경 맥락 포함)
   ├── 오답 선지별 "왜 아닌지" 설명
   └── 관련 개념 연결 링크
```

### 7.2 Key Features

#### Tier 1: MVP 핵심 기능 (v1.0)

**F1. 기출문제 CBT 모의고사**
- 역대 기출문제(4회~최신)를 실제 시험과 동일한 환경으로 제공
- 심화(50문항/70분) / 기본(50문항/70분) 별도 지원
- 문항 번호 네비게이션, 남은 시간 표시, 중간 저장
- 채점 후 즉시 결과 확인 (점수, 급수 판정, 시대별 정답률)

**F2. AI 오답노트 & 취약점 분석**
- 틀린 문제 자동 수집 및 분류 (시대별, 유형별)
- 시대별 정답률 레이더 차트 시각화
- 취약 영역 Top 3 자동 도출 및 맞춤 복습 추천
- 반복 오답 패턴 감지 ("이 유형을 3번 연속 틀렸습니다")

**F3. AI 해설 생성**
- 정답 선지: 역사적 배경과 맥락을 포함한 상세 해설
- 오답 선지: "왜 이것이 답이 아닌지" 개별 설명
- 관련 사건/인물 연결 (선후 관계, 인과관계)
- 해설 품질 피드백 버튼 ("도움이 됐어요" / "정확하지 않아요")

**F4. 급수별 학습 로드맵**
- 목표 급수 + 시험일 기반 자동 학습 플랜 생성
- 하루 권장 학습량 제시 (문제 수 + 예상 소요시간)
- 진행률 추적 및 자동 조정 (진도가 늦으면 플랜 재조정)

#### Tier 2: 차별화 기능 (v1.1)

**F5. 일일 퀴즈 & 연속 학습 스트릭**
- 매일 5~10문제 퀴즈 (취약 영역 가중 출제)
- 연속 학습일 기록 및 시각적 보상 (배지, 레벨업)
- 푸시 알림으로 학습 리마인더

**F6. D-day 스마트 알림**
- 시험일 카운트다운
- 시험 전 주차별 학습 가이드 푸시
- "오늘의 문제" 알림

**F7. 오프라인 모드**
- 문제 DB 로컬 저장으로 인터넷 없이 문제 풀이 가능
- 온라인 복귀 시 학습 기록 자동 동기화

#### Tier 3: 부가 기능 (v2.0+)

**F8. 한국사 타임라인 인터랙티브 맵**
- 시대별 주요 사건을 시각적 타임라인으로 탐색
- 사건 간 인과관계 연결선 표시

**F9. 연표 순서 맞추기 게임**
- 사건 카드 드래그&드롭 배열 (순서 배열 문제 유형 대비)

**F10. 사진/사료 문제 은행**
- 유물 사진, 사료 지문 등 시각 자료 기반 문제 별도 분류

| **로컬 저장소** | AsyncStorage | 사용자 프로필, 학습 기록, 오답노트 | ✅ 구현 완료 |
| **문제 DB** | 구조화된 JSON 파일 (번들) | 시대, 유형, 난이도, 급수별 태깅 | ✅ 구현 완료 |
| **이미지 저장소** | Cloudflare R2 (S3 호환) | 문제 이미지 CDN 서빙 | ✅ 구현 완료 |
| **저작도구** | Express + Vite + React (단일 프로세스) | 문제/시험 관리, AI 생성, PDF 추출 | ✅ 구현 완료 |
| **AI 생성** | Gemini API | 문제 생성, 이미지 생성, PDF 텍스트 추출 | ✅ 구현 완료 |
| **PDF 이미지 추출** | Gemini 3.1 Pro + PyMuPDF | PDF 지문/자료 영역 자동 감지·크롭·R2 업로드 | ✅ 구현 완료 |
| **AI 해설** | Claude API (Haiku/Sonnet) | 한국어 역사 지식 정확도, 비용 효율 | 🔲 미구현 |
| **백엔드/인증** | 서버 + 소셜 로그인 | 학습 기록 동기화, 계정 관리 | 🔲 미구현 |
| **오프라인** | SQLite (로컬 DB) | 문제 데이터 로컬 캐싱 | 🔲 미구현 |
| **푸시 알림** | Firebase Cloud Messaging | iOS/Android 통합 푸시 | 🔲 미구현 |
| **분석** | Mixpanel 또는 Amplitude | 사용자 행동 추적, 퍼널 분석 | 🔲 미구현 |

### 7.4 Assumptions

| # | 가정 | 검증 상태 | 위험도 |
|---|------|----------|--------|
| A1 | 기출문제를 상업적으로 활용 가능 (공공누리) | 미검증 | 🔴 Critical |
| A2 | AI 해설 정확도 90% 이상 달성 가능 | 미검증 | 🔴 High |
| A3 | 사용자가 월 3,000~5,000원 지불 의향 있음 | 미검증 | 🔴 High |
| A4 | 맞춤형 분석이 핵심 차별화로 작용함 | 미검증 | 🟡 Medium |
| A5 | 1~2인 개발팀이 3개월 내 MVP 출시 가능 | 미검증 | 🟡 Medium |
| A6 | 2027년 9급 대체로 응시자 수 증가 | 높은 확신 | 🟢 Low |

> **중요**: A1(기출문제 저작권)은 프로젝트 진행 전 반드시 먼저 확인해야 한다. 이것이 불가능하면 제품 전략 전체를 재검토해야 한다.
