**Scope**: MVP Tier 1 기능 (F1~F4) + 온보딩
**관련 문서**: [PRD](./PRD-korean-history-exam-app.md) | [Discovery Plan](./discovery-plan.md)

> - 🔲 미구현: US-18~22 (AI 해설), US-25~26 (진행률/재조정), US-27~28 (계정/결제)

## Epic 1: 온보딩 & 목표 설정

### US-01. 목표 급수 선택

**Title:** 목표 급수 선택

**Description:** 신규 사용자로서, 앱을 처음 열었을 때 목표 급수를 선택하고 싶다. 그래야 내 수준에 맞는 문제와 학습 콘텐츠를 받을 수 있다.

**Design:** PRD 7.1 온보딩 플로우 참고

**Acceptance Criteria:**
1. 앱 최초 실행 시 온보딩 화면이 자동으로 표시된다.
2. 시험 유형을 선택할 수 있다: "심화 (1·2·3급)" 또는 "기본 (4·5·6급)".
3. 각 유형의 간단한 설명이 표시된다 (예: "심화 — 공무원, 공기업 지원 시 필요").
4. 선택 후 "다음" 버튼을 누르면 시험일 설정 화면으로 이동한다.
5. 선택하지 않으면 다음 단계로 진행할 수 없다.

---

### US-02. 시험일(D-day) 설정

**Title:** 시험일 D-day 설정

**Description:** 신규 사용자로서, 목표 시험 회차의 날짜를 설정하고 싶다. 그래야 시험까지 남은 시간에 맞춰 학습 플랜을 받을 수 있다.

**Design:** PRD 7.1 온보딩 플로우 참고

**Acceptance Criteria:**
1. 예정된 시험 회차 목록이 표시되고, 탭 하나로 선택할 수 있다 (예: "제78회 — 2026.08.08").
2. 목록에 없는 경우 캘린더에서 직접 날짜를 선택할 수 있다.
3. 과거 날짜는 선택할 수 없다.
4. "아직 정하지 않았어요"를 선택하면 시험일 없이 진행할 수 있다.
5. 설정 후 D-day 카운트다운이 홈 화면에 표시된다.

---

### US-03. 하루 학습 시간 설정

**Title:** 일일 학습 시간 목표 설정

**Description:** 신규 사용자로서, 하루에 한국사 공부에 쓸 수 있는 시간을 설정하고 싶다. 그래야 내 생활 패턴에 맞는 학습량을 추천받을 수 있다.

**Design:** PRD 7.1 온보딩 플로우 참고

**Acceptance Criteria:**
1. 3가지 선택지가 제공된다: "10분 (가볍게)", "20분 (적당히)", "30분 (집중적으로)".
2. 각 선택지에 대략적인 하루 풀이 문항 수가 함께 표시된다 (예: "10분 — 약 5~8문제").
3. 하나를 선택하면 다음 단계로 진행된다.
4. 설정한 값은 마이페이지에서 언제든 변경할 수 있다.

---

### US-04. 진단 테스트

**Title:** 현재 수준 진단 테스트

**Description:** 신규 사용자로서, 짧은 진단 테스트를 풀어 내 현재 실력을 확인하고 싶다. 그래야 어디서부터 공부해야 할지 알 수 있다.

**Design:** PRD 7.1 온보딩 — 진단 테스트 (20문항, 5분) 참고

**Acceptance Criteria:**
1. 시대별 균등 배분된 20문항이 출제된다 (선사~현대, 시대당 2~3문항).
2. 제한 시간은 5분이며, 화면 상단에 남은 시간이 표시된다.
3. 시간이 초과되면 풀지 않은 문제는 오답 처리되고 자동 제출된다.
4. "건너뛰기"를 선택하면 진단 테스트 없이 홈 화면으로 이동할 수 있다.
5. 완료 후 시대별 정답률 요약이 표시되고 "학습 시작하기" 버튼이 나타난다.
6. 진단 결과는 취약점 분석의 초기 데이터로 활용된다.

---

### US-05. 홈 화면 대시보드

**Title:** 메인 홈 화면

**Description:** 응시자로서, 앱을 열면 오늘 할 학습, 시험까지 남은 일수, 내 진행 상황을 한눈에 보고 싶다. 그래야 바로 학습을 시작할 수 있다.

**Design:** PRD 7.1 메인 홈 와이어프레임 참고

**Acceptance Criteria:**
1. D-day 카운트다운이 화면 상단에 표시된다 (시험일 미설정 시 비표시).
2. "오늘의 학습" 카드에 추천 문제 수와 오답 복습 문제 수가 표시된다.
3. "학습 시작하기" 버튼을 탭하면 추천 학습 세션이 즉시 시작된다.
4. "나의 실력" 카드에 전체 정답률(%)과 취약 시대 Top 2가 요약 표시된다.
5. "CBT 모의고사" 카드에 가장 최근 모의고사 점수가 표시된다.
6. 하단 탭 바에 홈, 학습, 분석, MY 4개 탭이 존재한다.

## Epic 2: 기출문제 CBT 모의고사 (F1)

### US-06. 모의고사 회차 선택

**Title:** CBT 모의고사 회차 선택

**Description:** 응시자로서, 풀고 싶은 기출문제 회차를 직접 선택하고 싶다. 그래야 특정 시험을 집중적으로 대비할 수 있다.

**Design:** PRD 7.2 F1 참고

**Acceptance Criteria:**
1. 학습 탭에서 "CBT 모의고사" 메뉴를 탭하면 회차 목록이 표시된다.
2. 회차 목록은 최신순으로 정렬되며, 회차 번호와 시행일이 함께 표시된다.
3. 각 회차에 "심화" / "기본" 라벨이 표시된다.
4. 이미 풀었던 회차는 최근 점수가 함께 표시된다 (예: "82/100").
5. 무료 사용자는 최근 10회차까지 접근 가능하며, 나머지는 잠금 표시와 함께 "프리미엄" 배지가 보인다.
6. 회차를 탭하면 모의고사 시작 확인 화면이 표시된다.

---

### US-07. CBT 모의고사 풀기

**Title:** 실전 환경 모의고사 풀이

**Description:** 응시자로서, 실제 시험과 동일한 환경(50문항, 70분)에서 모의고사를 풀고 싶다. 그래야 실전 감각을 익힐 수 있다.

**Design:** PRD 7.1 CBT 모의고사 와이어프레임 참고

**Acceptance Criteria:**
1. 시작 시 선택한 급수 유형에 맞는 50문항이 표시된다.
2. 화면 상단에 남은 시간(70분)이 mm:ss 형식으로 표시되며 실시간 카운트다운된다.
3. 현재 문항 번호와 전체 문항 수가 "Q. 12 / 50" 형식으로 표시된다.
4. 4개 선지 중 하나를 탭하여 답을 선택할 수 있고, 선택한 선지에 체크 표시가 된다.
5. "이전" / "다음" 버튼으로 문항 간 이동이 가능하다.
6. 하단 문항 번호 네비게이터에서 답변한 문항(●)과 미답변 문항(○)이 구분 표시된다.
7. 문항 번호를 직접 탭하면 해당 문항으로 바로 이동한다.
8. 이미 선택한 답을 다른 선지를 탭하여 변경할 수 있다.
9. 사진/사료 자료가 포함된 문항은 자료가 문제 텍스트 아래에 표시된다.

---

### US-08. 모의고사 중간 저장 & 이어풀기

**Title:** 모의고사 중간 저장

**Description:** 응시자로서, 모의고사 풀이 중 앱을 종료해도 진행 상태가 저장되길 원한다. 그래야 나중에 이어서 풀 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 앱을 백그라운드로 전환하거나 종료해도 현재까지의 답안과 남은 시간이 자동 저장된다.
2. 앱 재실행 시 "진행 중인 모의고사가 있습니다. 이어서 풀까요?" 팝업이 표시된다.
3. "이어서 풀기"를 선택하면 마지막으로 풀던 문항과 남은 시간부터 재개된다.
4. "처음부터 다시 풀기"를 선택하면 해당 회차가 초기화되고 70분이 새로 시작된다.
5. 저장된 진행 상태는 최대 1건만 유지된다 (새 모의고사 시작 시 기존 진행 상태 덮어쓰기 경고).

---

### US-09. 모의고사 시간 초과 자동 제출

**Title:** 시간 초과 시 자동 제출

**Description:** 응시자로서, 70분이 지나면 자동으로 답안이 제출되길 원한다. 그래야 실제 시험처럼 시간 관리 연습을 할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 남은 시간이 5분 이하가 되면 타이머 색상이 빨간색으로 변경된다.
2. 남은 시간이 1분 이하가 되면 짧은 진동 알림이 한 번 발생한다.
3. 시간이 0:00이 되면 풀지 않은 문항은 오답 처리되고 자동 제출된다.
4. 자동 제출 후 채점 결과 화면으로 바로 이동한다.
5. "시간 전에 제출하기" 버튼을 탭하면 미답변 문항 수를 안내하는 확인 팝업이 표시된다.

---

### US-10. 모의고사 채점 결과 확인

**Title:** 모의고사 채점 결과

**Description:** 응시자로서, 모의고사를 제출하면 즉시 점수와 합격 여부를 확인하고 싶다. 그래야 현재 내 수준을 파악할 수 있다.

**Design:** PRD 7.1 결과 & 분석 플로우 참고

**Acceptance Criteria:**
1. 제출 즉시 채점이 완료되고 결과 화면이 표시된다.
2. 총점이 "82 / 100" 형식으로 크게 표시된다.
3. 급수 판정 결과가 표시된다 (예: "1급 합격 — 80점 이상").
4. 시대별 정답률이 막대 그래프로 시각화된다 (선사시대, 삼국, 남북국, 고려, 조선 전기, 조선 후기, 근대, 현대).
5. 유형별 정답률이 표시된다 (정치, 경제, 사회, 문화).
6. "오답 문제 보기" 버튼을 탭하면 틀린 문제 목록으로 이동한다.
7. "AI 해설 보기" 버튼을 탭하면 첫 번째 문항의 해설 화면으로 이동한다.

---

### US-11. 단원별 학습 모드

**Title:** 시대별/주제별 문제 풀이

**Description:** 응시자로서, 특정 시대나 주제만 골라서 문제를 풀고 싶다. 그래야 취약한 영역을 집중 공략할 수 있다.

**Design:** PRD 7.1 학습 모드 — 단원별 학습 참고

**Acceptance Criteria:**
1. 학습 탭에서 "단원별 학습"을 탭하면 시대 목록이 표시된다.
2. 시대 목록: 선사시대·고조선, 삼국시대, 남북국시대, 고려, 조선 전기, 조선 후기, 근대, 현대.
3. 각 시대 항목에 해당 시대의 총 문제 수와 나의 정답률이 함께 표시된다.
4. 시대를 선택하면 해당 시대의 문제가 순서대로 출제된다.
5. 단원별 학습에는 시간제한이 없다.
6. 문제를 풀 때마다 즉시 정답/오답 여부가 표시된다 (CBT와 달리 즉시 피드백).

## Epic 3: AI 오답노트 & 취약점 분석 (F2)

### US-12. 오답 자동 수집

**Title:** 오답 문제 자동 수집

**Description:** 응시자로서, 내가 틀린 문제가 자동으로 오답노트에 쌓이길 원한다. 그래야 따로 정리하지 않아도 틀린 문제를 모아볼 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. CBT 모의고사, 단원별 학습에서 틀린 문제가 자동으로 오답노트에 추가된다.
2. 오답노트는 분석 탭에서 접근할 수 있다.
3. 같은 문제를 여러 번 틀려도 오답노트에는 1건만 유지되며, 틀린 횟수가 기록된다.
4. 오답노트에서 각 문제에는 시대, 유형, 틀린 날짜, 틀린 횟수가 표시된다.
5. 이전에 틀렸지만 이후 맞힌 문제는 "해결됨" 라벨이 붙되, 목록에서 제거되지는 않는다.

---

### US-13. 오답 복습 모드

**Title:** 오답 문제 재도전

**Description:** 응시자로서, 오답노트에 있는 문제를 다시 풀어보고 싶다. 그래야 같은 실수를 반복하지 않을 수 있다.

**Design:** PRD 7.1 학습 모드 — 오답 복습 참고

**Acceptance Criteria:**
1. 오답노트에서 "오답 복습 시작" 버튼을 탭하면 오답 문제 풀이가 시작된다.
2. 기본적으로 틀린 횟수가 많은 문제가 우선 출제된다.
3. 시대별/유형별 필터를 적용하여 특정 영역의 오답만 복습할 수 있다.
4. 복습에서 문제를 맞히면 해당 문제에 "해결됨" 라벨이 추가된다.
5. 복습 완료 후 "N문제 중 M문제를 맞혔습니다" 요약이 표시된다.

---

### US-14. 시대별 취약점 분석 대시보드

**Title:** 시대별 취약점 분석

**Description:** 응시자로서, 어느 시대가 약한지 한눈에 파악하고 싶다. 그래야 공부할 순서를 정할 수 있다.

**Design:** PRD 7.1 결과 & 분석 — 시대별/유형별 정답률 차트 참고

**Acceptance Criteria:**
1. 분석 탭에 시대별 정답률이 레이더 차트(방사형 차트)로 시각화된다.
2. 8개 시대 축이 표시된다: 선사·고조선, 삼국, 남북국, 고려, 조선 전기, 조선 후기, 근대, 현대.
3. 정답률이 낮은 시대일수록 차트 안쪽으로 오므라들어 시각적으로 취약점이 드러난다.
4. 차트 아래에 "취약 영역 Top 3"가 텍스트로 표시된다 (예: "1. 근대 — 정답률 42%").
5. 각 취약 영역을 탭하면 해당 시대의 단원별 학습으로 바로 이동할 수 있다.
6. 데이터가 충분하지 않으면 (풀이 문제 수 < 20) "더 많은 문제를 풀면 정확한 분석이 가능합니다" 안내가 표시된다.

### US-15. 유형별 정답률 분석

**Title:** 문제 유형별 정답률

**Description:** 응시자로서, 정치·경제·사회·문화 유형 중 어떤 유형에 약한지 알고 싶다. 그래야 유형별로 대비 전략을 세울 수 있다.

**Design:** PRD 7.1 결과 & 분석 참고

**Acceptance Criteria:**
1. 분석 탭에서 시대별 차트 아래에 유형별 정답률이 막대 그래프로 표시된다.
2. 4개 유형이 표시된다: 정치, 경제, 사회, 문화.
3. 각 유형의 정답률(%)과 풀이 문제 수가 함께 표시된다.
4. 유형을 탭하면 해당 유형의 오답 문제 목록으로 이동한다.

### US-16. 반복 오답 패턴 알림

**Title:** 반복 오답 경고

**Description:** 응시자로서, 같은 유형의 문제를 반복해서 틀리면 알려주길 원한다. 그래야 의식적으로 해당 영역에 집중할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 같은 시대+유형 조합에서 3문제 이상 연속 오답 시 알림 카드가 표시된다.
2. 알림 메시지 예시: "고려시대 정치 문제를 3번 연속 틀렸어요. 집중 복습이 필요합니다."
3. 알림 카드에 "바로 복습하기" 버튼이 포함되며, 탭하면 해당 영역의 오답 복습이 시작된다.
4. 알림은 문제 풀이 직후 결과 화면에서 표시된다 (푸시 알림 아님).

### US-17. 맞춤 복습 추천

**Title:** AI 기반 다음 학습 추천

**Description:** 응시자로서, 분석 결과에 기반하여 "다음에 무엇을 공부하면 좋을지" 추천받고 싶다. 그래야 뭘 해야 할지 고민하는 시간을 줄일 수 있다.

**Design:** PRD 7.1 결과 & 분석 — "다음 학습 추천" 자동 생성 참고

**Acceptance Criteria:**
1. 모의고사 결과 화면과 분석 탭에 "추천 학습" 섹션이 표시된다.
2. 취약 시대 + 유형을 조합하여 최대 3개 추천을 생성한다 (예: "고려시대 정치 문제 10개 풀기").
3. 각 추천에 예상 소요시간이 함께 표시된다 (예: "약 8분").
4. 추천을 탭하면 해당 영역의 문제 풀이가 바로 시작된다.
5. 풀이 이력이 없거나 부족하면 진단 테스트 또는 최신 회차 모의고사를 추천한다.

## Epic 4: AI 해설 생성 (F3)

### US-18. 정답 해설 보기

**Title:** AI 정답 해설

**Description:** 응시자로서, 문제를 풀고 나면 정답에 대한 상세한 해설을 읽고 싶다. 그래야 단순 암기가 아니라 맥락을 이해할 수 있다.

**Design:** PRD 7.1 AI 해설 와이어프레임 참고

**Acceptance Criteria:**
1. 문제 풀이 후 "해설 보기" 버튼을 탭하면 AI 해설 화면이 표시된다.
2. 정답 선지가 상단에 강조 표시된다 (예: "정답: ③ 아관파천").
3. 해설에는 해당 사건의 역사적 배경, 시기, 의미가 포함된다.
4. 해설 길이는 200자 내외로 간결하게 제공된다.
5. 해설 하단에 관련 사건 연결이 표시된다 (예: "을미사변 → 아관파천 → 대한제국 수립").
6. 해설 로딩 중에는 스켈레톤 UI 또는 로딩 인디케이터가 표시된다.

---

### US-19. 오답 선지별 해설

**Title:** 오답 선지 개별 해설

**Description:** 응시자로서, 내가 고른 오답이 왜 틀렸는지 각 선지별로 알고 싶다. 그래야 비슷한 선지에서 다시 헷갈리지 않을 수 있다.

**Design:** PRD 7.1 AI 해설 와이어프레임 참고

**Acceptance Criteria:**
1. 정답 해설 아래에 오답 선지 각각에 대한 설명이 표시된다.
2. 각 오답 선지 옆에 "왜 아닌지" 이유가 한 줄 요약으로 표시된다 (예: "① 갑신정변(1884) → 시기 불일치").
3. 오답 선지를 탭하면 해당 선지에 대한 추가 상세 설명이 펼쳐진다 (아코디언).
4. 사용자가 선택했던 오답 선지는 빨간색으로 강조 표시된다.

---

### US-20. 해설 품질 피드백

**Title:** AI 해설 피드백 수집

**Description:** 응시자로서, AI 해설이 도움이 되었는지 피드백을 남기고 싶다. 그래야 해설 품질이 지속적으로 개선될 수 있다.

**Design:** PRD 7.2 F3 — 해설 품질 피드백 버튼 참고

**Acceptance Criteria:**
1. 해설 하단에 "도움이 됐어요 👍" / "정확하지 않아요 👎" 두 개 버튼이 표시된다.
2. 버튼을 탭하면 피드백이 즉시 기록되고 "피드백 감사합니다" 토스트가 표시된다.
3. "정확하지 않아요"를 탭한 경우 선택 사항으로 간단한 사유를 입력할 수 있다 (텍스트 필드, 최대 200자).
4. 한 문제에 대해 피드백은 1회만 제출할 수 있고, 이미 제출했으면 선택한 버튼이 비활성화된다.
5. 피드백 데이터는 서버에 저장되어 해설 개선에 활용된다.

---

### US-21. 해설 간 네비게이션

**Title:** 문제별 해설 이동

**Description:** 응시자로서, 모의고사 해설을 문제 순서대로 넘기면서 읽고 싶다. 그래야 모든 문제의 해설을 빠짐없이 확인할 수 있다.

**Design:** PRD 7.1 해설 와이어프레임 — [다음 문제] 버튼 참고

**Acceptance Criteria:**
1. 해설 화면 하단에 "이전 문제" / "다음 문제" 버튼이 있다.
2. 첫 번째 문항에서는 "이전 문제" 버튼이 비활성화되고, 마지막 문항에서는 "다음 문제"가 비활성화된다.
3. "오답만 보기" 토글이 있어 틀린 문제의 해설만 순서대로 볼 수 있다.
4. 현재 문항 번호가 "12 / 50" 형식으로 표시된다.

---

### US-22. 프리미엄 해설 접근 제한

**Title:** 무료/프리미엄 해설 분리

**Description:** 무료 사용자로서, 일부 문제의 AI 해설을 무료로 볼 수 있되, 전체 해설은 프리미엄에서 제공됨을 알고 싶다. 그래야 유료 전환 여부를 판단할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 무료 사용자는 각 모의고사에서 처음 5문항의 AI 해설을 무료로 볼 수 있다.
2. 6번째 문항부터 해설 영역에 블러 처리와 함께 "프리미엄으로 전체 해설 보기" 버튼이 표시된다.
3. 버튼을 탭하면 구독 안내 화면으로 이동한다.
4. 프리미엄 사용자는 모든 문항의 해설을 제한 없이 볼 수 있다.

## Epic 5: 급수별 학습 로드맵 (F4)

### US-23. 자동 학습 플랜 생성

**Title:** 목표 기반 학습 플랜 자동 생성

**Description:** 응시자로서, 목표 급수와 시험일을 기반으로 자동으로 학습 플랜을 받고 싶다. 그래야 매일 뭘 공부해야 하는지 고민하지 않아도 된다.

**Design:** PRD 7.2 F4 참고

**Acceptance Criteria:**
1. 온보딩 완료 후 (목표 급수 + 시험일 + 학습 시간 설정 시) 자동으로 학습 플랜이 생성된다.
2. 학습 플랜은 시험일까지 남은 기간을 주 단위로 나누어 구성된다.
3. 각 주차에 학습할 시대/주제와 권장 문제 수가 표시된다.
4. 진단 테스트를 풀었다면 취약 시대에 더 많은 시간이 배정된다.
5. 시험일이 설정되지 않은 경우 "4주 기본 플랜"이 기본값으로 생성된다.

---

### US-24. 일일 학습량 표시

**Title:** 오늘의 학습량 확인

**Description:** 응시자로서, 오늘 풀어야 할 문제 수와 예상 소요시간을 확인하고 싶다. 그래야 시간을 미리 확보하고 계획적으로 공부할 수 있다.

**Design:** PRD 7.1 메인 홈 — 오늘의 학습 카드 참고

**Acceptance Criteria:**
1. 홈 화면 "오늘의 학습" 카드에 오늘 풀어야 할 추천 문제 수가 표시된다.
2. 예상 소요시간이 함께 표시된다 (예: "15문제 · 약 12분").
3. 오답 복습 문제가 있으면 별도로 표시된다 (예: "오답 복습 8개").
4. 오늘의 학습을 모두 완료하면 체크 표시와 함께 "오늘 학습 완료!" 메시지가 표시된다.

---

### US-25. 학습 진행률 추적

**Title:** 전체 진행률 확인

**Description:** 응시자로서, 학습 플랜 대비 내 진행 상황을 한눈에 보고 싶다. 그래야 진도가 밀리고 있는지 알 수 있다.

**Design:** PRD 7.1 메인 홈 — 학습 진행률 바 참고

**Acceptance Criteria:**
1. 홈 화면에 전체 학습 플랜 대비 진행률이 프로그레스 바로 표시된다 (예: "████████░░ 76%").
2. 진행률은 완료한 학습 세션 수 / 전체 계획된 세션 수로 계산된다.
3. 분석 탭에서 주차별 계획 vs 실제 학습량을 비교하는 차트를 볼 수 있다.
4. 진도가 계획 대비 70% 미만이면 "조금 서두르세요!" 안내가 표시된다.

---

### US-26. 학습 플랜 자동 재조정

**Title:** 플랜 자동 재조정

**Description:** 응시자로서, 진도가 늦어지면 남은 기간에 맞게 플랜이 자동 조정되길 원한다. 그래야 현실적인 계획으로 공부할 수 있다.

**Design:** PRD 7.2 F4 — 진행률 추적 및 자동 조정 참고

**Acceptance Criteria:**
1. 주 1회 (월요일) 플랜 대비 실제 진행률을 체크하고, 필요시 자동 재조정된다.
2. 재조정 시 남은 기간에 미완료 학습량을 재배분한다.
3. 재조정 발생 시 "학습 플랜이 업데이트되었습니다" 알림 카드가 홈 화면에 표시된다.
4. 알림 카드에 변경 내용 요약이 표시된다 (예: "이번 주 일일 학습량이 15개 → 20개로 조정되었습니다").
5. 사용자가 원하면 마이페이지에서 목표 급수, 시험일, 학습 시간을 수동으로 변경하여 플랜을 재생성할 수 있다.

## Epic 6: 사용자 계정 & 설정

### US-27. 회원가입 & 로그인

**Title:** 회원가입 및 로그인

**Description:** 신규 사용자로서, 간편하게 가입하고 로그인하고 싶다. 그래야 학습 기록이 저장되고 기기를 바꿔도 이어서 공부할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 소셜 로그인을 지원한다: 카카오, 네이버, Apple (iOS), Google.
2. 이메일 회원가입도 지원한다 (이메일 + 비밀번호).
3. 비회원 모드로도 앱 사용이 가능하되, "학습 기록은 이 기기에만 저장됩니다" 안내가 표시된다.
4. 로그인 후 학습 기록, 오답노트, 분석 데이터가 서버에 동기화된다.
5. 비회원 상태에서 가입하면 기존 학습 기록이 계정에 연동된다.

---

### US-28. 프리미엄 구독

**Title:** 프리미엄 구독 결제

**Description:** 무료 사용자로서, 프리미엄 구독의 혜택을 확인하고 결제하고 싶다. 그래야 AI 해설, 전체 기출문제 등 모든 기능을 사용할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 구독 안내 화면에 무료 vs 프리미엄 기능 비교표가 표시된다.
2. 가격 옵션이 표시된다: 월 3,900원 / 연 29,900원.
3. 연간 구독에 "월 2,492원 — 36% 할인" 계산 결과가 표시된다.
4. 결제는 앱스토어/구글플레이 인앱결제로 진행된다.
5. 7일 무료 체험이 제공되며, 체험 기간 종료 전 알림이 발송된다.
6. 구독 상태는 마이페이지에서 확인 및 해지할 수 있다.

---

### US-29. 설정 변경

**Title:** 학습 설정 변경

**Description:** 응시자로서, 목표 급수, 시험일, 학습 시간을 나중에 변경하고 싶다. 그래야 상황이 바뀌었을 때 플랜을 업데이트할 수 있다.

**Design:** N/A

**Acceptance Criteria:**
1. 마이(MY) 탭에서 "학습 설정" 메뉴를 탭하면 설정 화면이 표시된다.
2. 목표 급수(심화/기본), 시험일, 하루 학습 시간을 각각 변경할 수 있다.
3. 설정 변경 후 "저장" 버튼을 누르면 학습 플랜이 자동 재생성된다.
4. 재생성 전 "학습 플랜이 초기화됩니다. 진행하시겠습니까?" 확인 팝업이 표시된다.
5. 기존 학습 기록(오답노트, 모의고사 결과)은 설정 변경 시에도 유지된다.
