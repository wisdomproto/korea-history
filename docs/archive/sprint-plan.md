# Sprint Plan: 한국사 마스터 MVP (v1.0)

**Product**: 한국사 마스터
**Sprint 주기**: 2주 (10 영업일)
**팀 구성**: 1인 개발 (풀스택)
**총 기간**: 12주 (Sprint 1~6) + 사전 준비 2주
**관련 문서**: [PRD](./PRD-korean-history-exam-app.md) | [User Stories](./user-stories-mvp.md)

> **현재 상태 (2026-03-08)**: Sprint 4까지의 핵심 기능 대부분 구현 완료. UI/UX 리뉴얼(적응형+모던 디자인) 완료.
> 실제 개발은 스프린트 순서와 무관하게 진행됨 — 아래는 원래 계획이며, 각 항목에 실제 완료 상태를 표시함.
