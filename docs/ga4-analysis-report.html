왜 절대값이 낮은가 — 가설
  • A
    iframe 자동 로드 안 함 — YouTube embed가 lazy load라 사용자가 스크롤·클릭 전엔 iframe 자체가 없을 수 있음. 노트 페이지에서 영상 위치 확인 필요.

  • D
    대안 가설 — 사용자가 영상보다 텍스트 해설을 선호. 이 경우 iframe 자체를 lazy하게 두고 "강의 영상 보기" 토글로 바꾸는 것도 옵션.
