1. 현대문 읽기 (구조·삽입·순서) 출제 106회 (절반)

국어 시험의 절반 이상. PSAT형 개편 이후 더욱 비중 증가. 5개 패턴만 마스터하면 50%는 보장.

패턴 1 — 글의 주제·요지 찾기

  • 전략: 첫 문단 + 마지막 문단 우선. 접속어 (그러나·따라서·결국)에서 결론 찾기
  • 함정: 너무 좁은 사례·예시는 주제 아님. 가장 포괄적인 진술 선택
  • 중심 화제: 글에 가장 자주 등장하는 개념

패턴 2 — 글의 순서 배열 (가)~(라)

  • 전략 1: 지시어(이·그·저)와 접속어(하지만·따라서·또한) 추적
  • 전략 2: 일반 → 구체, 원인 → 결과, 문제제기 → 해결 흐름 파악
  • 전략 3: 첫 문장 후보 2~3개 → 마지막 문장 → 중간 채우기
  • 대표 함정: 시간 순서가 아닌 논리적 순서

패턴 3 — 문장·문단 삽입 위치

  • 전략: 삽입할 문장의 지시어·접속어 분석 → 그 문장이 가리키는 대상이 직전에 와야 함
  • 예: "이러한 현상은…" → '현상'에 해당하는 내용 직후
  • 예: "그러나…" → 반대 내용 직후

패턴 4 — 빈칸 추론 (㉠·㉡·㉢)

  • 전략: 빈칸 앞뒤 문장의 논리 관계 (인과·대조·예시) 파악
  • 접속어가 단서: 따라서/그러므로 → 결론, 그러나/반면 → 대조
  • 여러 빈칸이 있으면 짝짓기 (각 빈칸의 단서 → 선지 매칭)

패턴 5 — 내용 일치·불일치

  • 전략: 선지 키워드 → 본문에서 직접 확인 (추론 금지)
  • 함정 5종: ① 부분 사실 + 일부 왜곡, ② 정도 과장 (모두/항상/절대), ③ 인과 역전, ④ 본문에 없는 정보 추가, ⑤ 비교 대상 바꾸기
출제 패턴: "글에 대한 이해로 적절하지 않은 것은?" — 4개 선지 중 3개는 본문 일치, 1개가 함정. 함정 유형 5종을 의식하면 정답률 급상승.

4. 현대문학 (시·소설·수필) 출제 13회

현대시

  • 화자(시적 자아): 시 안에서 말하는 사람. 시인 ≠ 화자
  • 운율: 외형률(정형시), 내재율(자유시·산문시)
  • 심상(이미지): 시각·청각·후각·미각·촉각·공감각
  • 주요 시인: 김소월(진달래꽃), 한용운(님의 침묵), 윤동주(서시), 정지용(향수), 백석(여승), 김수영(풀)

현대소설

  • 3요소: 인물·사건·배경
  • 시점: 1인칭 주인공/관찰자, 3인칭 전지적 작가/관찰자
  • 구성: 발단·전개·위기·절정·결말 (5단 구성)
  • 주요 작가: 김동인(감자), 현진건(운수 좋은 날), 염상섭(만세전), 채만식(태평천하), 이상(날개), 황순원(소나기), 박완서, 이청준

수필·희곡·시나리오

  • 수필: 형식 자유, 개성적, 1인칭, 비전문적
  • 희곡: 무대 상연 목적, 대사+지문, 갈등 중심
  • 시나리오: 영화 대본, S#(scene), F.I/F.O 등 영상 용어

5. 화법·작문 출제 9회

화법의 원리

  • 그라이스 협력의 원리: 양·질·관련성·태도
  • 공손성의 원리: 요령·관용·찬동·겸양·동의
  • 순서 교대: 대화의 기본 구조

토의 vs 토론

구분토의토론
목적문제 해결·합의 도출주장 관철·승패 가르기
참여자같은 입장에서 협력찬·반 대립
형태심포지엄, 패널, 포럼, 원탁입론·반론·교차질의·최종변론

작문 — 글쓰기 단계

  1. 계획하기: 주제·목적·예상 독자·매체 결정
  2. 내용 생성: 자료 수집·정리
  3. 내용 조직: 개요 작성, 짜임
  4. 표현하기: 초고 작성
  5. 고쳐쓰기: 수정·보완

설명문·논설문 짜임

  • 설명문: 처음(화제 제시) → 중간(설명) → 끝(요약·정리)
  • 논설문: 서론(문제제기) → 본론(주장+논거) → 결론(주장 강조)

6. 맞춤법·표준어·로마자 출제 8회

한글 맞춤법 핵심 규정

  • 제1항: "한글 맞춤법은 표준어를 소리대로 적되, 어법에 맞도록 함을 원칙으로 한다"
  • 두음법칙: 한자어 첫소리 'ㄴ·ㄹ' → 'ㅇ·ㄴ' (예: 녀자→여자, 락원→낙원)
  • 사이시옷: 순우리말 합성어, 뒷말 첫소리 된소리/ㄴ첨가/ㄴㄴ첨가 (예: 나뭇가지, 콧날)
  • 자주 틀리는 표기:
    • 되 vs 돼: '되어'로 풀어 자연스러우면 '돼' (예: 돼요, 안 돼)
    • 안 vs 않: '아니'면 '안', '아니하'면 '않' (예: 안 가다, 가지 않다)
    • 왠 vs 웬: '왜인지' 줄임은 '왠지', 그 외는 '웬'
    • 몇일 (X) → 며칠 (O)
    • 설겆이 (X) → 설거지 (O)

띄어쓰기

  • 조사는 앞말에 붙임 (예: 학교에서)
  • 의존명사는 띄움 (예: 먹을 것, 한 번)
  • 단위 명사는 띄움. 단 순서·숫자와 결합 시 붙임 가능 (예: 두 시 = 2시)
  • 보조용언은 원칙 띄어쓰기, 허용 붙임 (예: 도와 주다 = 도와주다)

표준어 규정

  • 표준어 사정 원칙: 교양 있는 사람들이 두루 쓰는 현대 서울말
  • 주요 표준어: 강낭콩(O)/강남콩(X), 깡총깡총(X)/깡충깡충(O), 미루나무(O)/미류나무(X)

로마자 표기

  • 국어의 로마자 표기법 (2000): 표준발음에 따라 적음
  • 예: 김치 = kimchi, 비빔밥 = bibimbap, 종로 = Jongno (자음동화 반영)
  • 이름은 성과 이름 띄어 씀 (예: Hong Gildong)

외래어 표기

  • 받침은 'ㄱ·ㄴ·ㄹ·ㅁ·ㅂ·ㅅ·ㅇ' 7개만
  • 파열음 표기는 된소리 쓰지 않음 (예: cafe = 카페 O, 까페 X)
  • 예: 초콜릿(O)/쵸콜렛(X), 카메라(O)/카메라(X), 바비큐(O)/바베큐(X)

7. 국어사·언어 일반 출제 2회

국어의 시대 구분

시대기간특징
고대국어~10세기한자 차용 (이두·향찰)
중세국어10~16세기고려·조선 전기. 훈민정음 창제 (1443)
근대국어17~19세기조선 후기. 음운·문법 단순화
현대국어20세기~표준어 정립

훈민정음 (1443 창제, 1446 반포)

  • 창제자: 세종대왕
  • 제자 원리: 자음 — 발음기관 모방 (ㄱㄴㅁㅅㅇ 기본 5자 + 가획) / 모음 — 천(·)·지(ㅡ)·인(ㅣ) 3재
  • 해례본: 28자 (자음 17 + 모음 11). 현재 24자 (자음 14 + 모음 10)
  • 훈민정음 해례본: 1997 유네스코 세계기록유산

언어의 일반적 특성

  • 자의성: 의미와 형식의 관계가 필연적이지 않음
  • 사회성: 사회적 약속 (개인이 마음대로 바꾸지 못함)
  • 역사성: 시간에 따라 변화
  • 창조성: 무한히 새로운 문장 생성
  • 분절성: 연속된 사실을 분절해 표현
  • 추상성: 개별 사물에서 공통 속성 추출

8. 고전문학 (참고) 출제 0~소수
