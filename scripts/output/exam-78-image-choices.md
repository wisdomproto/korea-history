## 수동 보강 옵션 (선택 사항)

각 image-choice Q에 대해 사이트 UX를 개선하려면:

1. **`choiceImages[]` 채우기** — Q9/Q14/Q26의 선지 합본을 5개 개별 PNG로 분할 → 각 URL을 `choiceImages` 배열에 넣기. 그러면 사이트에서 선지 박스마다 그림이 따로 표시됨.
2. **현재 텍스트 선지 유지** — Gemini가 추출한 텍스트 설명을 선지로 활용 (학생이 그림과 텍스트 매칭). 지금 상태.

분할 작업은 저작도구의 이미지 업로드 기능으로 5장 별도 업로드 후 `choiceImages: [url1..url5]`에 넣으면 됨. 자동 분할 스크립트가 필요하면 알려주세요.
