1. 전략 개요 및 현황 분석

서비스 핵심 셀링 포인트 (오픈채팅용)

  • 완전 무료 + 회원가입 불필요 — 링크만 공유하면 즉시 사용 가능 (전환 마찰 제로)
  • 40~77회 기출 1,900+ 문항 — 가장 방대한 무료 기출 DB
  • AI 해설 + 최태성 강의 자동 연결 — 문제별 YouTube 타임스탬프 1,850개
  • 요약노트 87개 — 시대별 정리 + 관련 기출 즉시 풀기
  • 맞춤형 학습 — 시대/유형/키워드 선택 학습, 오답노트 자동 수집
핵심 인사이트: "무료 + 회원가입 없음"이라는 특성은 오픈채팅방 마케팅에서 최고의 무기입니다. 홍보해도 "광고"가 아닌 "정보 공유"로 인식됩니다. 유료 서비스나 앱 설치를 권유하는 것과는 본질적으로 다릅니다.

2. 기존 오픈채팅방 활용 전략

2.1 침투 단계별 로드맵

1단계: 관찰 & 분석 (1~2주차)

  • 7개 방 모두 입장 (같은 날 한꺼번에 들어가지 말것 — 2~3일 간격)
  • 2주간 메시지 읽기만 수행. 절대 발언하지 않음
  • 파악 항목: 주요 활동 시간대, 자주 나오는 질문 유형, 핵심 활동 멤버, 방장 성향, 홍보 허용 분위기
  • 각 방별 관찰 노트 작성 (활동량, 분위기, 한능검 관련 질문 빈도)

2단계: 전문가 포지셔닝 (3~4주차)

  • 한능검 관련 질문에 정성스러운 텍스트 답변 제공 (링크 없이)
  • "몇 급 필요해요?" → 자격 요건 정리해서 답변
  • "어디서부터 공부해야 해요?" → 시대별 공부 순서 + 팁 제공
  • "기출 어디서 풀어요?" → 아직 링크 공유 안 함. "기출 많이 풀어보세요" 정도만
  • 주 3~5회 이상 답변 → "한능검은 저 분한테 물어보면 됨" 인식 형성

3단계: 자연스러운 링크 공유 시작 (5~6주차)

  • 누군가 기출 관련 질문을 하면 답변 + "여기서 바로 풀어볼 수 있어요" 형태로 링크 추가
  • 반드시 질문에 대한 답변을 먼저 쓴 후 보조 자료로 링크 첨부
  • 처음에는 특정 문제 링크 (gcnote.co.kr/exam/XX/YY) — 메인 페이지가 아닌 구체적 링크
  • 주 1~2회로 제한. 매번 링크 붙이면 스팸 인식

4단계: 정보 제공자 정착 (7주차~)

  • 시험 시즌에 맞춘 유용한 정보 + 링크를 정기적으로 공유
  • "다음 시험 D-14입니다! 최근 3회 기출 많이 나온 키워드 정리" + 링크
  • 가답안 발행 시 가장 먼저 공유 (가답안 기능 구현 시)
  • 멤버들이 먼저 링크를 공유해주는 단계가 목표

2.2 방별 맞춤 접근법

한능검 Q&A (1,150명) — 직접 타겟

접근: 가장 적극적으로 활동. 한능검 전문가로 포지셔닝.

콘텐츠: 기출 해설, 시대별 핵심 정리, 공부법 팁, 시험 일정 안내

링크 공유 타이밍: "이 문제 해설이 잘 나와있는 곳" 형태로 특정 문제 페이지 공유

주의: 이 방은 경쟁 서비스도 활동할 가능성 높음. 노골적 홍보 절대 금지, 순수 도움 자세 유지

경찰공무원 + 경찰최대 (3,808명) — 최대 규모

접근: 한능검은 경찰 시험의 "부수 과목". 본업(경찰 시험) 존중하면서 한능검 서포터 역할.

콘텐츠: "경찰 시험 준비하시면서 한능검은 어떻게 하세요?" 대화 유도

핵심 메시지: "한능검은 따로 시간 쓰지 말고 기출만 반복하면 됩니다" → 기출 사이트 자연 연결

시즌 활용: 한능검 시험 전 "경찰 준비하시는 분들, 이번 한능검 접수 기간이에요!" 알림 역할

소방/세무/간호/계리직 (4,838명) — 안정적 보조 채널

접근: 각 직군별 한능검 필요 급수를 정확히 알려주는 "한능검 가이드" 역할

콘텐츠: "소방 시험 한능검 2급 커트라인 알려드릴게요", "세무직은 한능검 몇 급 필요해요?" 등

빈도: 주 1~2회 활동. 경찰 방보다 낮은 빈도 (한능검 비중이 상대적으로 낮음)

2.3 절대 하지 말아야 할 것 (레드라인)

  • 입장 직후 링크 공유 — 즉시 강퇴 + 스팸 신고 가능
  • "기출노트 한능검 좋아요!" 식의 직접 홍보 — 광고로 인식
  • 같은 링크 반복 공유 — 3회 이상 같은 URL은 스팸
  • 다른 사람 질문 무시하고 링크만 투척 — 텍스트 답변 없이 링크만 던지지 말 것
  • 여러 방에 동시에 같은 메시지 복붙 — 발각 시 전체 신뢰 상실
  • 방장/운영자와 갈등 — 방장이 불편해하면 즉시 중단

4. 자연스러운 홍보 메시지 설계

4.1 상황별 메시지 템플릿

아래 템플릿은 기존 오픈채팅방에서 사용하는 것으로, 반드시 해당 질문/상황이 발생했을 때만 사용합니다.

상황 1: "한능검 기출 어디서 풀어요?"
기출은 많이 풀수록 좋아요! 최근 회차부터 역순으로 푸는 걸 추천해요. 저는 gcnote.co.kr 여기서 풀고 있는데 무료에 회원가입도 없어서 편해요. 77회부터 40회까지 다 있고, 문제마다 해설도 나와요. 근데 중요한 건 풀고 나서 틀린 문제 복습하는 거예요! 같은 유형이 계속 나오거든요.

포인트: 답변(공부법 팁)이 메인, 링크는 보조. 마지막에 추가 팁으로 마무리.

  • 각 방별 관찰 노트 작성

2주차: 나머지 방 입장 + 관찰 계속

  • 2순위 4개 방 입장: 소방, 세무, 간호, 계리직
  • 7개 방 모두 관찰 지속. 아직 발언하지 않음
  • 각 방의 한능검 관련 질문 빈도와 유형 정리

3~4주차: 답변 활동 시작

  • 1순위 방부터 한능검 관련 질문에 텍스트 답변 시작
  • 주 3~5회 이상 양질의 답변 제공 (링크 없이)
  • "한능검 잘 아는 사람" 인식 형성

Phase 2: 링크 공유 + 자체 방 개설 (5~8주차)

5~6주차: 자연스러운 링크 공유 시작

  • 1순위 방에서 답변 시 관련 페이지 링크 첨부 시작
  • 주 1~2회, 반드시 텍스트 답변과 함께
  • 반응 관찰: 클릭율, 방 분위기 변화, 방장 반응

7주차: 자체 "한능검 기출 스터디방" 개설

  • 방 이름: "한능검 기출 스터디방 | 기출노트"
  • 환영 메시지 + 규칙 설정
  • 웹사이트 Footer에 오픈채팅 링크 추가
  • "오늘의 기출 1문제" 운영 시작 (매일 평일)

8주차: 초기 멤버 확보

  • 웹사이트 유입 + 기존 방 활동에서 자연 유도
  • 시험 결과 페이지에 "스터디방 참여하기" CTA 추가
  • 목표: 50명 돌파

Phase 3: 안정화 + 시험 시즌 (9~12주차)

9~10주차: 콘텐츠 루틴 안정화

  • 주간 캘린더 기반 콘텐츠 루틴 정착
  • 기존 방 링크 공유 빈도 최적화 (반응 기반 조정)
  • 자체 방 멤버 질문 대응 프로세스 안정화

11~12주차: 시험 시즌 집중 실행

  • 다음 시험 D-14부터 시즌 전술 실행
  • "2주 완성 기출 챌린지" 운영
  • 기존 방 + 자체 방 동시 시즌 콘텐츠 배포
  • 시험 후 가답안 속보 공유
  • KPI 리뷰 + 전략 보정

핵심 원칙 요약:
  1. "도움 먼저, 홍보 나중" — 모든 활동의 기본 원칙. 답변이 메인, 링크는 보조.
  2. "무료 + 회원가입 없음"이 최강 무기 — 링크를 공유해도 "광고"가 아닌 "정보 공유"로 인식되는 유일한 조건.
  3. 시험 시즌에 집중 — 비시즌은 최소 유지, 시즌 D-14부터 전력 투구. 연 5회 시험이 5번의 대규모 유입 기회.
  4. 기존 방(단기) + 자체 방(장기) 병행 — 기존 방에서 신뢰 쌓고, 자체 방으로 충성 사용자 모으는 이중 트랙.
  5. 선순환 루프 구축 — 오픈채팅 → 웹사이트 → 오픈채팅 → 웹사이트. 이 루프가 자동으로 돌면 성공.
