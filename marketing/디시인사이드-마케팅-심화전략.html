1. DC 플랫폼 이해
왜 DC인가, 그리고 왜 위험한가

1-1. 기출노트 한능검이 DC에 적합한 이유

핵심: DC에서 가장 욕 안 먹는 홍보는 "진짜 무료 + 진짜 유용한" 정보 공유다. 우리는 완전 무료, 회원가입 없음, 결제 유도 없음 — DC에서 "이거 홍보 아니냐?" 물어왔을 때 "무료인데 뭘 홍보함?"으로 끝나는 구조다.
우리의 강점 (DC 적합)왜 강점인가
완전 무료"돈 받는 것도 아닌데 홍보는 무슨" → 의심 해소의 결정타
회원가입 없음개인정보 수집 안 한다 → "팔아먹을 것도 없잖아" 논리 성립
실제로 유용함1,900+ 문항, AI 해설, 요약노트 87개 → 써보면 진짜 좋음
공시생/취준생 대다수가 필요한능검은 가산점 필수 → DC 공시생 갤러리와 완벽 매칭
경쟁 서비스 대부분 유료"다른 앱은 다 결제 유도하는데 여기는 진짜 무료" 비교 우위

1-2. DC 특수 문화 — 반드시 숙지

DC에서 통하는 것
  • 단점 먼저 언급하고 장점 나중에
  • 짧고 거친 말투 (반말/존반말)
  • 핸드폰 직찍 스크린샷 (저화질 OK)
  • "모르는 사람 많던데 알려줌" 식 정보 공유
  • 물어봤을 때만 URL 알려주기
  • 갤러리 내부 밈/용어 자연스럽게 사용
  • "ㅋㅋ" "ㄹㅇ" "ㅇㅇ" 등 줄임말
DC에서 죽는 것
  • 장점만 나열하는 칭찬글
  • 정중한 존댓말 (기업 계정 냄새)
  • 고퀄리티 홍보 이미지/배너
  • "추천합니다" "꼭 써보세요" 식 권유
  • 본문에 URL 바로 박기
  • 밈 모르면서 억지로 쓰기
  • "광고 아닌데" "홍보 아니고" 변명

2-2. Tier 1 — 핵심 갤러리 심층 분석

공무원 갤러리 (공갤) — "전장" 난이도: 최상

문화 특성

  • 분위기: 공시 스트레스 + 냉소 + 자조. "공시생 현타" 글이 일상. 살벌함
  • 말투: 거친 반말. "~함" "~임" 체. 존댓말 쓰면 외부인 취급
  • 금기: "공시 쉽다" "나는 금방 붙었다" 식 자랑 → 즉시 공격
  • 밈/용어: "합격각" "탈공" "N수생" "현타" "이번 시험 폭탄" 등
  • 글 밀림 속도: 매우 빠름 (분 단위)
  • 홍보 감지력: 극도로 높음. 조금이라도 의심되면 "홍보충 아웃"

진입 전략

  • 최소 2~3주 일반 활동 필수 (공시 관련 잡담, 공감 댓글)
  • "한국사 가산점 준비 어떻게 함?" 질문으로 자연스럽게 한능검 화제 꺼내기
  • 절대 첫 글에서 사이트 언급 금지 → 3~4번째 글부터 자연 언급

맞춤 톤앤매너

좋은 예: "한능검 기출 풀다가 77회 근현대 존나 어려운 거 발견함ㅋㅋ 이 문제 맞은 사람 있냐?"
나쁜 예: "한능검 공부하기 좋은 사이트를 발견했습니다. 무료이고 정말 유용하니 한번 방문해보세요!"

추천 바이럴 유형 (우선순위)

  1. 짧은 댓글 — "한능검 기출 어디서 풂?" 질문에 답변 (가장 안전)
  2. 빡침 해결 — "한능검 앱마다 결제 유도하는 거 개빡침" (공감 유도)
  3. 시험 직전 — 시험 2주 전 "벼락치기 루트" 정보글
자격증 갤러리 (자갤) — "정보의 장" 난이도: 중상

문화 특성

  • 분위기: 다양한 자격증 정보 교류. 공갤보다 부드럽지만 여전히 DC
  • 말투: 반말/존반말 혼용. "~함" "~인데" "~거든" 체
  • 특징: 정보글에 대한 수용도가 높은 편. 유용하면 추천 잘 해줌
  • 밈/용어: "자격증 뽑기" "가성비 자격증" "유효기간" "난이도 체감" 등
  • 주의: 한능검 외에도 다양한 자격증 화제 → 한능검 글이 너무 많으면 의심

진입 전략

  • 한능검 외 다른 자격증 관련 댓글도 달아야 자연스러움
  • "자격증 몇 개 따봤는데 한능검은 이렇게 했음" 식 경험 공유
  • 정보글 형태가 가장 환영받는 갤러리 → "가성비 정보" 유형 최적

맞춤 톤앤매너

좋은 예: "한능검 가성비로 따려면 기출 반복이 답임. 무료 사이트에서 3회독 돌렸더니 1급 나옴"
나쁜 예: "한능검 사이트 추천! gcnote.co.kr에서 무료로 공부할 수 있어요~"

추천 바이럴 유형 (우선순위)

  1. 가성비 정보 — "한능검 0원으로 따는 법" (자갤 최적)
  2. 합격 후기 — "2주 독학 1급" (실전 정보 + 간접 언급)
  3. 좌표 공유 — "(좌표) 한능검 기출 무료 사이트" (정보글에 관대한 갤)
취업 갤러리 (취갤) — "생존의 장" 난이도: 최상

문화 특성

  • 분위기: 취준 스트레스 + 독설 + 자조. 공갤만큼 살벌
  • 말투: 거친 반말. 비꼬기/냉소 일상. 긍정적인 글 오히려 의심
  • 특징: 스펙, 자소서, 면접 관련 글이 주류. 한능검은 "스펙 부분"
  • 밈/용어: "스펙충" "취준 몇 년차" "서류 탈락" "합스" "취업 현타" 등
  • 주의: "쉽게 됐다" 뉘앙스 절대 금지. 고생한 느낌 필수

진입 전략

  • 취업 관련 일반 글/댓글로 2~3주 활동
  • "스펙 만들기" 맥락에서 한능검을 자연스럽게 등장시킴
  • "가성비 스펙"으로 프레이밍 — 취준생이 돈 없다는 공감 포인트 활용

맞춤 톤앤매너

좋은 예: "취준하면서 돈 없어서 한능검이라도 무료로 따려고 기출 돌렸는데 가성비 ㄹㅇ임"
나쁜 예: "취업 준비에 유용한 한능검 학습 사이트를 알려드립니다!"

추천 바이럴 유형 (우선순위)

  1. 빡침 해결 — "한능검 앱마다 돈 내라는 거" (취준생 돈 없음 공감)
  2. 가성비 정보 — "0원 스펙 만들기" (취갤 핵심 관심사)
  3. 짧은 댓글 — 스펙 질문에 간결 답변
공시생 미니갤 — "동료의 방" 난이도: 중

문화 특성

  • 분위기: 공갤보다 부드러움. 실제 공시 준비생끼리 정보 공유 + 위로
  • 말투: 반말 기본이지만 공갤보다 덜 공격적. "~인데" "~더라" 체
  • 특징: 소규모라 같은 사람이 자주 보임 → 활동 이력이 쌓이면 신뢰 빠르게 형성
  • 장점: 정보 공유에 대한 감사 표현이 공갤보다 많음

진입 전략

  • 1~2주 일반 활동 후 정보글 가능
  • "같이 준비하는 사람"으로 포지셔닝 → 공부법 나누기
  • 미니갤은 글이 천천히 밀려서 노출 시간 김

추천 바이럴 유형 (우선순위)

  1. 합격 후기 — "한능검 이렇게 해서 1급 땄음" (동료 톤)
  2. 가성비 정보 — "돈 안 드는 한능검 공부법" (공시생 예산 공감)
  3. 시험 직전 — 시험 전 벼락치기 정보

2-3. Tier 2 — 직접 타겟 갤러리

한능검 마이너갤 + 한능검 미니갤 — "홈그라운드" 난이도: 낮음

문화 특성

  • 분위기: 한능검 직접 타겟. 시험 정보, 기출 분석, 합격 후기 중심
  • 말투: DC 기본 반말이지만, 학습 관련이라 비교적 온화
  • 특징: 활동량 적음 → 글이 오래 살아있음 → 노출 시간 길다
  • 장점: 한능검 학습 자료 공유가 자연스러운 유일한 갤러리. 의심 받을 확률 가장 낮음
  • 주의: 소규모라 같은 사람이 반복 글 쓰면 바로 티남

진입 전략

  • 등업 거의 필요 없음 (소규모 갤러리 특성)
  • 기출 분석글, 시험 후기, 공부법 등 일반 글 3~5개 후 자연 언급
  • "요즘 여기서 기출 풀고 있는데" 식으로 사이트 자연스럽게 등장

추천 바이럴 유형 (우선순위)

  1. 좌표 공유 — 여기서는 직접 공유가 가장 자연스러움
  2. 합격 후기 — 기출 사이트 언급하며 공부법 공유
  3. 시험 직전 — 벼락치기 루트 + 사이트 추천
공무원스터디 미니갤 + 공무원합격 미니갤 — "스터디룸" 난이도: 중

문화 특성

  • 스터디 미니갤: 공부 인증, 스터디 모집, 공부법 공유. 학습 자료 공유에 관대
  • 합격 미니갤: 합격 후기, 수기 중심. 합격자 조언에 대한 수용도 높음
  • 말투: 존반말~반말. 서로 도와주는 분위기가 있음
  • 장점: "공부법 공유" 맥락에서 사이트 언급이 매우 자연스러움

추천 바이럴 유형

  • 스터디갤: 가성비 정보 + 합격 후기
  • 합격갤: 합격 후기 (가장 적합) + 좌표 공유

2-4. Tier 3 — 직렬 니치 갤러리 (6개)

기술직 / 교정직 / 교행현직 / 공무원 / 사회복지직 / 시설관리직 난이도: 낮음~중

공통 문화 특성

  • 분위기: 해당 직렬 준비생/현직 위주. 소규모. 직렬 특화 정보 중심
  • 말투: 갤러리마다 다르지만, 대체로 미니갤은 온화한 편
  • 핵심 어프로치: "해당 직렬 한국사 가산점" 맥락으로 진입
  • 특징: 한능검을 "메인 관심사"가 아닌 "필요한 가산점"으로 다룸

3. 등업 전략 — 갤러리 유형별 로드맵
DC 계정을 "신뢰할 수 있는 갤러"로 육성하는 단계별 가이드
대원칙: 등업 기간에 사이트 언급은 절대 금지. 이 기간은 "계정의 자연스러움"을 만드는 투자 기간이다. 조급하면 망한다.

3-1. 계정 준비 (Day 0)

항목권장주의
계정 수2~3개 (1개만 쓰면 차단 시 대안 없음)같은 IP로 동시 다중 활동 금지
닉네임DC스러운 평범한 닉네임 (예: 한능검준비중, 9급도전, ㅇㅇ)"기출노트" "gcnote" 등 사이트 연상 닉네임 절대 금지
가입 메일네이버/지메일 (새 메일 권장)회사 메일, 사이트 도메인 메일 금지

3-2. 갤러리 유형별 등업 로드맵

A. 정식 갤러리 (공무원갤, 자격증갤, 취업갤, 기술직갤)

Week 1 — 관찰 + 첫 활동
  • 갤러리 글 50개 이상 읽기 → 분위기, 말투, 밈 파악
  • 일반 글에 공감 댓글 5~7개 (예: "ㅇㅇ ㅋㅋ 공감", "ㄹㅇ 나도 그럼")
  • 질문글에 도움 댓글 2~3개 (한능검 무관한 일반 질문)
  • 홍보/사이트 언급: 절대 금지
Week 2 — 일반 게시글 작성
  • 해당 갤러리 주제에 맞는 일반 글 2~3개 작성
  • 예: 공갤 → "9급 일반행정 경쟁률 ㅎㄷㄷ" / 자갤 → "올해 딸 자격증 리스트"
  • 댓글 달리면 자연스럽게 소통 (무시하지 않기)
  • 홍보/사이트 언급: 절대 금지
Week 3 — 한능검 화제 자연 진입
  • 한능검 관련 질문글에 도움 답변 (사이트 없이 순수 정보만)
  • "한능검 공부법 질문" → "기출 3회독이 답임. 최근 5개년 위주로"
  • 한능검 관련 자체 질문글도 OK ("한능검 77회 20번 이거 답 뭐임?")
  • 사이트 언급: 아직 금지. "한능검 잘 아는 사람" 이미지만 쌓기
Week 4+ — 자연스러운 첫 언급
  • 누군가 "기출 어디서 풀어?" 질문 → 댓글로 자연 답변
  • 또는 "기출 풀다가 발견한 것" 글에서 간접 언급
  • URL은 본문 아닌 댓글에서, 물어봤을 때만
  • 이후 주 1~2회 간격으로 자연스러운 활동 유지

B. 마이너/미니 갤러리 (한능검갤, 공시생미니갤, 직렬갤 등)

Week 1 — 관찰 + 댓글 활동
  • 기존 글 읽고 분위기 파악. 소규모라 문화가 빠르게 파악됨
  • 공감/도움 댓글 3~5개
  • 마이너/미니갤은 활동 자체가 적어서 댓글만 달아도 눈에 띔
Week 2 — 정보글 + 간접 언급
  • 소규모 갤러리는 2주면 충분. 정보글에 대한 거부감 낮음
  • 한능검 마이너/미니갤의 경우 → 1주 만에 자연 언급 가능
  • 직렬갤은 "가산점 정보" 맥락에서 자연스럽게

4-4. 유형 3: 가성비 정보 — 상세 가이드

작성 원칙

  • "돈 안 쓰고 한능검 따는 법" 프레이밍 — 정보 가치를 극대화
  • gcnote만 추천하면 홍보 냄새 → 다른 무료 자료도 함께 언급 (유튜브 강의 등)
  • "실제 내 경험" 기반으로 작성 — 추상적 정보가 아닌 구체적 루트
  • 자격증갤과 취업갤에서 가장 잘 먹히는 유형

자갤/취갤 버전

제목: 한능검 1급 돈 0원으로 따는 루트 정리

한능검 교재 사지 마셈. 기출이 답임

1단계: 기출 3회독
gcnote.co.kr에서 최근 5개년(73~77회) 기출 3번 돌림
무료고 가입 없이 바로 풀 수 있음
틀리면 관련 요약노트로 연결되고 최태성 강의도 바로 뜸

2단계: 약한 시대 집중
맞춤형 학습 기능으로 약한 시대만 골라서 반복
나는 근현대가 약해서 거기만 100문제 돌림

3단계: 오답 복습
틀린 문제 자동으로 모아주는 기능 있어서 시험 전날 오답만 쭉 돌림

이거로 2주 만에 1급 땄음. 교재비 0원.
다른 무료 자료: 최태성 1TV 유튜브 (강의), EBS 한능검 (기본 개념)
근데 기출 반복이 제일 중요하고 위에 사이트가 그게 가장 잘 됨

4-5. 유형 4: 합격 후기 — 상세 가이드

작성 원칙

  • "이렇게 해서 붙었음" 경험담 형태 — 가장 자연스러운 사이트 언급 구조
  • 점수까지 구체적으로 (예: "82점 1급") → 신뢰도 상승
  • "금방 했다" 보다 "고생했지만 이렇게 해서 됐다" 톤이 DC에서 안전
  • 공시생미니갤, 합격미니갤, 한능검갤에서 가장 적합

공시생미니갤/합격미니갤 버전

제목: 한능검 1급 후기 (독학 3주, 무료 자료만)

공시 준비하면서 한국사 가산점 챙겨야 해서 한능검 봤음
돈 없어서 교재 안 사고 무료 자료만 썼는데 82점 1급 나옴

뭘 썼냐면:
기출 → gcnote.co.kr (40~77회 전부, 무료, 가입없음)
강의 → 최태성 1TV 유튜브

이 사이트가 좋았던 게:
- 틀리면 관련 요약노트로 바로 넘어감 (따로 안 찾아도 됨)
- 해설 밑에 최태성 강의 해당 부분부터 재생 (시간 절약 개쩜)
- 오답 자동 수집 → 시험 전날 틀린 거만 돌림

공부법:
1주차: 최근 5개년 기출 1회독
2주차: 약한 시대 집중 + 2회독
3주차: 오답 복습 + 3회독

시간 없으면 2주도 가능할 듯. 기출 반복이 답이었음

4-6. 유형 5: 짧은 댓글 — 상세 가이드

이 유형이 가장 중요하다. 글보다 댓글이 DC에서 훨씬 자연스럽고 안전하다. 바이럴 활동의 60~70%는 댓글로 해야 한다.

5-1. URL 관리

최악의 시나리오: gcnote.co.kr URL 자체가 갤러리 금지어로 등록되면, 해당 갤러리에서 영구적으로 URL 사용이 불가능해진다. 이건 복구가 안 된다.
전략설명위험도
URL 최소 노출 본문에 URL 직접 박지 않기. "기출노트 한능검 검색하면 나옴" 또는 댓글에서만 공유 안전
사이트명으로 우회 "gcnote" "기출노트" 같은 사이트명만 쓰고 URL은 물어보면 알려주기 안전
검색 유도 "한능검 기출 무료 검색하면 나옴" → 네이버/구글 검색으로 유도 매우 안전
URL 직접 게시 (본문) 한능검 전용 갤러리에서만 제한적으로. Tier 1 갤러리에서는 자제 주의
URL 직접 게시 (댓글) 질문에 대한 답변으로만. 본문보다는 안전하지만 빈도 관리 필수 보통

URL 차단 시 대응 플랜

단계대응
1차: URL 차단됨"기출노트 한능검" 검색 유도로 전환. URL 언급 완전 중단
2차: 사이트명도 필터링"한능검 무료 사이트" 같은 일반적 표현으로 전환. 직접 DM 추천
3차: 해당 갤러리 포기차단이 심한 갤러리는 포기하고 다른 갤러리에 집중

5-2. 문체 차단 방지

차단 패턴예시대응
동일 문구 복붙 14개 갤러리에 같은 글 복붙 갤러리마다 말투, 구조, 내용을 다르게. 완전히 다시 작성
정형화된 추천 패턴 "이 사이트 추천합니다. 장점은 1... 2... 3..." 개인 경험 기반. 스토리텔링 형태로. 장점 나열 대신 에피소드
과도한 정보 나열 "1,900문항, AI 해설, 87개 요약노트, 오답노트..." 한 번에 1~2개 기능만 언급. 전체 기능 목록 나열은 홍보물
지나치게 긍정적 "이건 진짜 미쳤음 완벽함 안 쓰면 손해" 단점 먼저 + 장점 나중. "이건 좀 그런데, 이건 괜찮음" 톤
기업 용어 사용 "사용자 경험" "학습 효율" "콘텐츠" "플랫폼" DC 용어로 교체: "쓰기 편함" "공부 잘 됨" "자료" "사이트"

5-3. 패턴 차단 방지

위험 패턴탐지 원리회피 방법
같은 시간대 반복 게시 매일 같은 시간에 글 올리면 봇/업무로 의심 게시 시간을 랜덤하게. 점심/저녁/밤 번갈아가며
짧은 간격 다중 갤 활동 30분 내 3개 갤러리에 글 → 다중 갤 홍보 의심 같은 날 다른 갤러리 활동 시 최소 2~3시간 간격
새 글 직후 자추/자댓 글 쓰자마자 다른 계정으로 추천/댓글 다른 계정의 반응은 최소 6시간 이후. 자추는 절대 하지 않기
특정 URL만 반복 공유 계정의 모든 글/댓글에 같은 URL 활동 비율 7:3 유지. 사이트 무관한 활동이 다수여야 함

5-4. "홍보충" 의심 시 대응 시나리오

시나리오 1: "홍보 아니냐?"
  • 안 좋은 대응: "홍보 아닙니다! 정말 좋아서 추천하는 거예요"
  • 좋은 대응: "ㅋㅋ 홍보면 이렇게 안 씀. 무료인데 뭘 홍보함"
  • 최선의 대응: 무시. 다른 댓글에만 반응
시나리오 2: 집요하게 추궁
  • "이 계정 글 다 사이트 홍보잖아" 식 공격
  • 대응: 절대 반박하지 않음. 해당 갤러리에서 사이트 언급 2주 이상 중단
  • 다른 주제로 일반 활동 → 시간이 지나면 잊혀짐
시나리오 3: 계정 차단
  • 대응: 해당 갤러리 같은 계정으로 재시도 절대 금지
  • 다른 갤러리에 다른 계정으로 이동
  • 차단 사유 분석 → 다른 계정에서 같은 실수 반복하지 않기

8. 위기 대응 매뉴얼
최악의 시나리오별 대응 프로토콜

8-1. 위기 등급 분류

등급상황대응 속도
Level 1 개별 댓글에서 "홍보 아니냐?" 의심 가볍게 대응 또는 무시. 일상적 상황
Level 2 한 갤러리에서 "이 계정 홍보계정임" 글 올라옴 즉시 해당 갤러리 활동 중단. 2주간 잠수
Level 3 여러 갤러리에 "gcnote 홍보계정 조심" 퍼짐 전 갤러리 활동 즉시 중단. 최소 1개월 쿨다운
Level 4 gcnote.co.kr URL이 갤러리 금지어로 등록 검색 유도 방식으로 전환. URL 언급 영구 중단

8-2. Level별 상세 대응

Level 1 대응
  • 가볍게 넘기기: "ㅋㅋ 홍보면 돈이라도 받지" 또는 무시
  • 절대 길게 해명하지 않기
  • 다음 활동까지 3~5일 간격 두기
  • 이후 해당 갤러리에서 일반 활동 비율 높이기
Level 2 대응
  • 해당 갤러리 2주간 완전 잠수 (일반 활동도 중단)
  • 해당 계정으로 사이트 언급 1개월 금지
  • 다른 계정으로 다른 갤러리에서만 활동
  • 2주 후 해당 갤러리에 일반 활동만으로 복귀
Level 3 대응
  • 전 갤러리, 전 계정 활동 즉시 중단
  • 최소 1개월 쿨다운 기간
  • 기존 계정 폐기 고려 (오염된 계정)
  • 1개월 후 새 계정으로 처음부터 등업 시작
  • 복귀 시 활동 패턴 완전히 변경
Level 4 대응
  • URL 차단된 갤러리에서 URL 사용 영구 중단
  • "기출노트 한능검 검색" 또는 "한능검 무료 사이트 검색" 유도
  • 다른 갤러리에서도 URL 대신 검색 유도 방식 확대
  • 장기적으로 SEO 강화하여 검색 유도만으로 충분하게 만들기

8-3. 사이트 단점 지적 대응

지적 내용좋은 대응나쁜 대응
"앱 없어서 불편함" "ㅇㅇ 앱은 없어서 좀 그럼. 근데 웹에서도 되고 무료니까" "곧 앱이 출시될 예정입니다!"
"디자인 구림" "ㅋㅋ 디자인은 좀 그렇긴 한데 기능은 됨" "디자인이 그렇게 나쁘지 않은데요..."
"해설 틀린 거 있음" "AI 해설이라 가끔 그럴 수 있음. 근데 대부분 맞던데" "어떤 문제인지 알려주시면 수정하겠습니다"
"이미 다른 사이트 쓰는데" "ㅇㅇ 다른 거 쓰고 있으면 그거 쓰셈. 나는 여기가 편하더라" "우리 사이트가 더 좋은 이유는..."
핵심 원칙: "운영자"처럼 대응하면 정체가 들통난다. 항상 "그냥 쓰는 유저" 입장에서만 대응. 버그 리포트 받거나, 업데이트 예고하거나, 해명하는 순간 끝이다.
