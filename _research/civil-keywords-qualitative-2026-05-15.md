## 공무원 매핑 — Tier별 후보 + 신뢰도 등급

신뢰도 ⭐⭐⭐ = 한능검 패턴 직역 + 우리 콘텐츠 fit 100%
신뢰도 ⭐⭐ = 패턴 강함, 콘텐츠 보강 필요할 수도
신뢰도 ⭐ = 가능성 있으나 경쟁 셀 듯

## 콘텐츠 작업 우선순위 — 즉시 적용 가능 항목

1. **한국사 cross 메타 추가 (zero cost)**: `/9급/한국사` 라우트들 메타에 Tier S 키워드 박기. 이미 어제 작업한 `civil-seo.ts` 헬퍼에서 한국사 특화 키워드 ext에 "시대별 정리"·"흐름 정리" 들어가 있으니 자연스럽게 노출. 추가로 "단원별 기출"·"정리본" 보강 검토.
2. **전공 단권화 노트 발행 (이미 13개 본문 보유)**: 9급 행정법·행정학·형법 등 메타에 패턴 D 적용 — `boost-civil-9-exam-seo.mjs` 패턴 D 키워드에 이미 포함. 데이터 동기화만 하면 됨.
3. **Tier B 브랜드 확장**: layout.tsx default title에 "기출노트 공무원" 시그널 추가 검토 — 단 한능검 메인 dilution risk 측정 (옵션 A 결정 vs 통합 hub 전환 — `memory/project_civil_9_seo_strategy.md` 참조)
4. **Naver API 키 갱신 후 정량 재검증** ⭐ 블로커
5. **블로그 batch 12편 작성 검토** (한능검 22편 패턴 복제) — 황금키워드 long-form 글, 6~12주 색인 예상
